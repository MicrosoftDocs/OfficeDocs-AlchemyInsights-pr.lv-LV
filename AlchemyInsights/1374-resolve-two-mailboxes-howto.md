# <a name="a-user-has-two-mailboxes"></a>Lietotājam ir divas pastkastes

Hibrīdās vidēs, kas izmantot Azure Active Directory savienojums (AAD savienot) vai DirSync var nejauši izraisīt lietotājam ir divas pastkastes: viens lokālajā uzņēmumā un vienā mākonī.
  
Lai atrisinātu šo problēmu, skatīt [Pastāvīgi skaidrs iepriekšējo pastkastes Info](https://blogs.technet.microsoft.com/exchange/2018/01/17/permanently-clear-previous-mailbox-info/).
