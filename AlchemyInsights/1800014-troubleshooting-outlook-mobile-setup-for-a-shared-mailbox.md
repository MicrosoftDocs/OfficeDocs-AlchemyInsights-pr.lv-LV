# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>Outlook mobilo koplietojamās pastkastes iestatījumu problēmu novēršana

Nevar piekļūt koplietojuma pastkastes no Outlook app, kas ir instalēta mobilajā ierīcē. Vai vēlaties šo līdzekli? Ļaujiet Outlook inženieru komanda, kas zina. [Balsojums šeit](https://go.microsoft.com/fwlink/?linked=862116)!
  
Var izmantot tikai koplietojamās pastkastes uz mobilo ierīci, atveriet pārlūkprogrammu Office 365 pierakstīšanos un pārejiet uz Outlook Web. Instrukcijas skatiet [Add koplietojamo pastkasti programmā Outlook izveidojāt Web](https://support.office.com/en-us/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207). No Outlook Web jūs varēsiet piekļūt koplietojuma pastkastei.
  
Papildinformācija: [Palīdzība Outlook iOS un Android app,](https://support.office.com/en-us/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
