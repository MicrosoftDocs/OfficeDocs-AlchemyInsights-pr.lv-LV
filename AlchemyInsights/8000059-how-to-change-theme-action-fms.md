# <a name="how-to-change-your-theme"></a>Kā mainīt savu dizainu

Mainīt savu dizainu Outlook.com vai programmā Outlook Web, dodieties uz sadaļu **Iestatījumi**![iestatījumus](media/f4b2e798-fff1-4a14-931f-5677a4543b58.png) un izvēlieties motīvu saskaņā ar **tēmu**. Lai apskatītu visus pieejamos dizainus, atlasiet **Skatīt visu** saskaņā ar **tēmu**.
