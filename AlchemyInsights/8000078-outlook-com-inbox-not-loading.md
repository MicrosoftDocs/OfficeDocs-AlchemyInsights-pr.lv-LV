# <a name="inbox-not-loading-with-firefox"></a>Iesūtnē nav iekraušana ar Firefox

Dažas pievienojumprogrammas pārlūkprogrammā Firefox var neļaut iesūtnē Outlook.com iekraušanas.
  
Apmeklējiet Firefox atbalsts, lai uzzinātu, kā [atspējot vai noņemt pievienojumprogrammas](https://support.mozilla.org/kb/disable-or-remove-add-ons).
