# <a name="access-services"></a>Access pakalpojumi

SharePoint Online līdzeklis Access pakalpojumi ir novecojis. Jaunas Access tīmekļa programmas un tīmekļa datu bāzes vairs nevar izveidot, un pēc 2018 tam tiks beidzētas arī esošās tīmekļa lietojumprogrammas un tīmekļa datu bāzes. Šīs izmaiņas neietekmē Access datora datu bāzes (. accdb). Iesakām Microsoft PowerApps kā alternatīvu piekļuvei tīmekļa programmām un tīmekļa datu bāzēm. 
  
[Informācija par Microsoft PowerApps](https://powerapps.microsoft.com/)
