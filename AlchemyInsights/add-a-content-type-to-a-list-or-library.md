# <a name="add-a-content-type-to-a-list-or-library"></a>Satura tipa pievienošana sarakstam vai bibliotēkai

1. Atveriet [saraksta vai bibliotēkas iestatījumus](https://support.microsoft.com/en-us/office/edit-list-settings-in-sharepoint-online-4d35793b-246e-42a3-990c-563a83795b7f).
2. Sadaļā **satura tipi**atlasiet **Pievienot no esošiem vietnes satura tipiem**. Ja **satura tipi** neparādās, atlasiet **Papildu iestatījumi**.
3. Atlasiet sarakstā **Atlasīt vietnes satura tipus** no.
4. Sarakstā Pieejamie vietnes satura tipi atlasiet vajadzīgā satura tipa nosaukumu un pārvietojiet atlasīto satura tipu uz saraksta pievienotiem satura tipiem, atlasiet Pievienot.
5. Atkārtojiet darbības papildu satura tipiem.

Papildinformāciju skatiet sadaļā [satura tipa pievienošana sarakstam vai bibliotēkai](https://support.microsoft.com/en-us/office/add-a-content-type-to-a-list-or-library-917366ae-f7a2-47ad-87a5-9689a1884e60).
