# <a name="add-or-remove-an-email-address-for-a-user"></a>Lietotāja e-pasta adreses pievienošana vai noņemšana

Lai sāktu darbu, [veiciet tālāk norādītās darbības](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions) Microsoft 365 administrēšanas centrā.

 **Piezīme**. lietotājam var būt vairāk nekā viena e-pasta adrese, bet papildu  *e-pasta aizstājvārdus*  var izmantot tikai e-pasta saņemšanai. Ja vēlaties nosūtīt e-pasta ziņojumu ar aizstājvārdu, izveidojiet [koplietojamu pastkasti](https://docs.microsoft.com/microsoft-365/admin/email/create-a-shared-mailbox). Šādā veidā jūs saņemsit aizstājvārdu par pasta sūtīšanu un saņemšanu.
