# <a name="add-permissions"></a>Atļauju pievienošana

1. Izmantojiet administratora piekrišanas galapunktu, lai [lietojumprogrammai konfigurētu statiski pieprasīto atļauju sarakstu](https://docs.microsoft.com/azure/active-directory/develop/v2-permissions-and-consent#to-configure-the-list-of-statically-requested-permissions-for-an-application).
1. Ir pieejamas divas atļaujas, lai piešķirtu iespēju izveidot lietojumprogrammu reģistrācijas, no kurām katrai ir atšķirīga darbība. Lai iegūtu papildinformāciju, skatiet rakstu [lietojumprogrammas reģistrācijas atļaujas pielāgotām lomām Azure Active Directory](https://docs.microsoft.com/azure/active-directory/roles/custom-available-permissions).
1. [Lomu atļaujas](https://docs.microsoft.com/azure/active-directory/roles/permissions-reference#role-permissions): tabulas, kas apraksta specifiskās atļaujas Azure Active Directory katrai lomai. Dažām lomām var būt papildu atļaujas Microsoft pakalpojumos ārpus Azure Active Directory.
