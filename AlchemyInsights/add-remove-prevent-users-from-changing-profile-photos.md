# <a name="add-remove-or-prevent-users-from-changing-profile-photos"></a>Pievienot, noņemt vai neļaut lietotājiem mainīt profila fotoattēlus

- **Profila fotoattēlu pievienošana:** Profila fotoattēlus var pievienot administrators [Microsoft 365 administrēšanas centrā, Aktīvie lietotāji](https://admin.microsoft.com/Adminportal/Home?source=applauncher#/users) vai  [Azure Active Directory lietotāju pārvaldība](https://portal.azure.com/#blade/Microsoft_AAD_IAM/UsersManagementMenuBlade/AllUsers).  Ja opcija "mainīt fotoattēlu" nav redzama, pārliecinieties, vai šim lietotājam ir piešķirta licence. Lietotājam lietotāja profilā var pievienot vai mainīt fotoattēlus no jebkura Microsoft 365 pakalpojuma, ekrāna augšējā labajā stūrī noklikšķinot uz viņu iniciāļiem/fotoattēlu. Papildinformāciju par profila fotoattēla pievienošanu skatiet rakstā [sava profila fotoattēla pievienošana pakalpojumā Microsoft 365](https://support.office.com/article/add-your-profile-photo-to-office-365-2eaf93fd-b3f1-43b9-9cdc-bdcd548435b7).

- **Profila fotoattēlu noņemšana** Profila fotoattēlus var noņemt administrators [Azure Active Directory lietotāju vadībā](https://portal.azure.com/#blade/Microsoft_AAD_IAM/UsersManagementMenuBlade/AllUsers) vai lietotājs savā Microsoft Teams lietotāja profilā.

- **Profila fotoattēla izmaiņu bloķēšana:** Varat bloķēt fotoattēlu izmaiņas visiem Microsoft 365 *, pievienojot Outlook Web App politiku katram rakstam, [bloķējot fotoattēlus vai ierobežojot atļaujas, lai mainītu Microsoft 365 profila fotoattēlu](https://answers.microsoft.com/msoffice/forum/msoffice_o365admin-mso_manage/locking-photos-or-restricting-permissions-to/1d19ae4f-de5d-4c3d-a0ad-4b8b8ac32e3d).

* Lūdzu, ņemiet vērā, ka Microsoft Teams pašlaik neatbalsta Outlook Web App politiku, lai bloķētu fotoattēlu izmaiņas, bet plāno pievienot šī līdzekļa atbalstu agrās 2020.
