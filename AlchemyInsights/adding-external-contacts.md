# <a name="enable-external-communications-and-add-contacts"></a>Ārējo sakaru iespējošana un kontaktpersonu pievienošana

Programmā Skype darbam varat pievienot personas savam kontaktpersonu sarakstam, lai tā būtu tikai klikšķa attālumā. Katrai pievienotajai kontaktpersonai tiek piešķirta dalība vienā vai vairākās jūsu kontaktpersonu grupās. Papildinformāciju skatiet [rakstā kontaktpersonas pievienošana programmā Skype darbam](https://support.office.com/article/add-a-contact-in-skype-for-business-89338023-2adf-4f5c-90b6-f8b6f72fadd1) . 

Pārliecinieties, vai nomnieka administrators ir veicis darbības, lai [atļautu ārējo saziņu](https://docs.microsoft.com/skypeforbusiness/set-up-skype-for-business-online/allow-users-to-contact-external-skype-for-business-users).

Vai vēlaties ļaut lietotājiem sazināties ar Skype patērētāju lietotājiem? Skatiet rakstu [ļaut Skype darbam lietotājiem pievienot Skype kontaktpersonas](https://docs.microsoft.com/skypeforbusiness/set-up-skype-for-business-online/let-skype-for-business-users-add-skype-contacts).
