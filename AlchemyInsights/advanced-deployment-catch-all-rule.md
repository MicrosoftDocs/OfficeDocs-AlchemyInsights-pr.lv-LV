# <a name="deploy-office-apps"></a>Office programmu izvietošana

Izvietot Office lietojumprogrammas vairākiem lietotājiem, izmantojot Office izvietošanas rīks, Microsoft galapunkta konfigurācijas pārvaldnieks, kā daļa no operētājsistēmas diska attēlu vai koplietošanas datora gadījumā, izmantojot Windows Server ar attālās darbvirsmas pakalpojumu (RDS) vai termināļa pakalpojumus, apsveriet iespēju izmantot **Office 365 ProPlus izvietošanas padomnieks**soli pa solim.
  
Lai sāktu darbu, pārliecinieties, vai esat pierakstījies kā nomnieka administrators vai globālā admin un pēc tam noklikšķiniet šeit: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance).
