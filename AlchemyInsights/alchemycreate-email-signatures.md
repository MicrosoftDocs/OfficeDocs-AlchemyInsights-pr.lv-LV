# <a name="create-email-signatures"></a>E-pasta parakstu izveide

Skatiet tālāk norādītās tēmas, lai uzzinātu, kā izveidot e-pasta parakstus programmā Outlook.
  
- Izveidojiet e-pasta parakstu savām ziņām: [izveidojiet un pievienojiet parakstu ziņojumiem](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx) vai [izveidojiet un pievienojiet e-pasta parakstu programmā Outlook Web App](https://support.office.com/article/0f230564-11b9-4239-83de-f10cbe4dfdfc.aspx).
    
- Izveidojiet globālu e-pasta parakstu visiem ziņojumiem, kas tiek sūtīti no jūsu uzņēmuma (tiek dēvēta arī par atrunu): [organizācijas līmeņa ziņojumu atrunas, paraksti, kājenes vai galvenes](https://go.microsoft.com/fwlink/p/?linkid=391096).
