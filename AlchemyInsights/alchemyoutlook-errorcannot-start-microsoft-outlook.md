# <a name="outlook-error-cannot-start-microsoft-outlook"></a>Outlook kļūda: nevar startēt programmu Microsoft Outlook

Vai rodas problēmas saistībā ar Outlook 2016, Outlook 2013 vai Outlook 2010 vai saņemšanu "nevar startēt Microsoft Outlook. Nevar atvērt Outlook logu "kļūdu ziņojumi?
  
Vispirms izmantojiet [šajā rakstā](https://support.office.com/article/I-can-t-start-Microsoft-Outlook-2016-2013-or-2010-or-receive-the-error-Cannot-start-Microsoft-Office-Outlook-Cannot-open-the-Outlook-Window-d1f69da6-b333-4650-97bf-4d77bd7abb85) norādītās problēmu novēršanas darbības, lai palīdzētu novērst šīs startēšanas problēmas. 
  
Ietekmētajā datorā varat arī palaist [papildu diagnostikas rīku](https://aka.ms/SaRA-OutlookAdvDiagnostics) .
