# <a name="47500-access-denied-please-try-again-later"></a>4.7.500 piekļuve liegta, lūdzu, vēlāk mēģiniet vēlreiz

Šī kļūda rodas, ja e-pasta trafika apjoms no avota IP adrese pārsniedz ierobežojumu, pamatojoties uz avota IP adreses reputāciju (vai reputācijas trūkumu).

E-pasta bloķēšana no avota IP adreses beigsies stundas laikā. Ja avota IP adrese ir lokālais e-pasta serveris, kas pieder jums, pārbaudiet pasta plūsmas savienotāja konfigurāciju. Ja darbība turpinās ilgāk nekā stundu, sazinieties ar atbalsta dienestu, lai pieprasītu izņēmumu avota IP adresei.
