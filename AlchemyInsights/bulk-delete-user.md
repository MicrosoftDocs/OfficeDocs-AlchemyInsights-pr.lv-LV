# <a name="bulk-delete-user"></a>Lielapjoma dzēšanas lietotājs

Varat noņemt lielu skaitu lietotāju no grupas, izmantojot ar komatu atdalītu vērtību (CSV) failu lielapjoma dzēšanas lietotājiem. Lai dzēstu lielapjoma lietotājus, kas izmanto Azure AD portālu, skatiet rakstu [lielapjoma dzēšanas lietotāji Azure Active Directory](https://docs.microsoft.com/azure/active-directory/users-groups-roles/users-bulk-delete).
