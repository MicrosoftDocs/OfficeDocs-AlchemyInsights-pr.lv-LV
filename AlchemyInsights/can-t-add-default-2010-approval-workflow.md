# <a name="cant-add-default-2010-approval-workflow"></a>Nevar pievienot noklusējuma apstiprinājuma darbplūsma 2010

Microsoft SharePoint vietņu kolekcijas sarakstam vai bibliotēkai nevar pievienot globāli atkārtoti darbplūsmai (piemēram, "apstiprināšana - SharePoint 2010"). Apstiprinājuma darbplūsma var trūkt. Lai atrisinātu šo problēmu, skatiet [nevar pievienot darbplūsmu globāli atkārtoti koplietošanas vides SharePoint sarakstā vai bibliotēkā](https://support.microsoft.com/help/4467263/sharepoint-designer-2013-shows-empty-wfpub-library).

Vairāk info par darbplūsmas apstiprinājumiem, skatiet [viss par apstiprinājuma darbplūsmas](https://support.office.com/article/All-about-Approval-workflows-078C5A89-821F-44A9-9530-40BB34F9F742). 
 
Uzziniet vairāk par [Mūsdienu apstiprinājumiem plūsmā](https://flow.microsoft.com/blog/introducing-modern-approvals).
