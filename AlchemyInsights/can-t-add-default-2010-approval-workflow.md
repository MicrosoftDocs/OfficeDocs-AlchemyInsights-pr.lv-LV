# <a name="cant-add-default-2010-approval-workflow"></a>Nevar pievienot noklusējuma apstiprinājuma darbplūsma 2010

Microsoft SharePoint vietņu kolekcijas sarakstam vai bibliotēkai nevar pievienot globāli atkārtoti darbplūsmai (piemēram, "apstiprināšana - SharePoint 2010"). Apstiprinājuma darbplūsma var trūkt. Lai atrisinātu šo jautājumu, kas attiecas uz [koplietošanas vides SharePoint sarakstā vai bibliotēkā globāli atkārtoti darbplūsmu nevar pievienot](https://support.microsoft.com/help/4467263/sharepoint-designer-2013-shows-empty-wfpub-library).
