# <a name="change-the-font-size-in-a-message"></a>Fonta lieluma maiņa ziņojumā

1. Ja ziņojums jau ir uzrakstīts, atlasiet tekstu.
2. Atlasiet  **formatēšanas** ikonu zem ziņojuma (tā ir pirmā ikona).
3. Atlasiet  **fonta lieluma**  ikonu virs ziņojuma (tā ir septītā ikona).
4. Atlasiet lielumu.
