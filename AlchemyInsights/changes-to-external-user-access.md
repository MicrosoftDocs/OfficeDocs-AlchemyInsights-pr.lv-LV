# <a name="changes-to-external-user-access"></a>Izmaiņas ārējā lietotāja piekļuves

Sākot ar 2018. gada 23. martā, ārējie lietotāji vairs nevarēs piekļūt saturam, kas ir kopīgota ar visiem, visiem autentificētiem lietotājiem vai visi veidlapas lietotāji, pat ja viņi aicināja vai pievienot atļauto grupām. Tikai ārējiem lietotājiem būs iespēja piekļūt saturam, kas ir kopīgota ar lietotāja vai grupas, kurām pieder lietotājs. Ja vēlaties paturēt esošos iestatījumus, lūdzu, skatiet:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
