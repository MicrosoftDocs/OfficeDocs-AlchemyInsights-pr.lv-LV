# <a name="changes-to-external-user-access"></a>Izmaiņas ārējā lietotāju piekļuvē

Sākot ar 23. martu, 2018 ārējie lietotāji vairs nevarēs piekļūt saturam, kas ir kopīgots ar ikvienu, visiem autentificētajiem lietotājiem vai visiem veidlapu lietotājiem, pat tad, ja tie ir uzaicināti vai pievienoti pilnvarotām grupām. Ārējie lietotāji varēs piekļūt tikai tam saturam, kas ir kopīgots ar lietotāju, vai ar grupām, kurās šis lietotājs pieder. Ja vēlaties paturēt esošos iestatījumus, lūdzu, skatiet: https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
