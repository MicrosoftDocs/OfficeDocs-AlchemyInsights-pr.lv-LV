# <a name="changes-to-external-user-access"></a>Izmaiņas ārējā lietotāja piekļuves

Sākot no marts 23, 2018, ārējie lietotāji vairs nevarēs piekļūt saturam, kas ir kopīgots ar visiem, visiem autentificētajiem lietotājiem vai visiem veidlapu lietotājiem, pat ja tie ir uzaicināti vai pievienoti pilnvarotām grupām. Ārējie lietotāji varēs piekļūt tikai tādam saturam, kas ir kopīgots ar lietotāju vai grupām, kurām lietotājs pieder. Ja vēlaties saglabāt savus esošos iestatījumus, lūdzu, skatiet:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
