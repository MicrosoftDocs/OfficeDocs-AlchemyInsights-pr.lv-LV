# <a name="changing-public-folder-permissions"></a>Mainīšanas publiskās mapes

Publisko mapju atļaujas var mainīt lietotāji un administratori programmā Outlook. Administratori var arī kontrolēt atļaujas no Exchange administratora Center (VNK), rīkojoties šādi:
  
1. O365 portālā, izmantojot izvēlni kreisajā navigācijas rūtī naviģējiet uz **Admin centri** \> **apmaiņu**.
    
2. Naviģējiet uz **publiskām mapēm**.
    
3. Šeit varat mainīt atsevišķas publiskās mapes atļaujas, piešķirot drošības grupas atļaujas. Gala lietotājam, lai mainītu publiskās mapes atļaujas, lietotāju vajadzībām ir īpašnieka tiesības uz mapi.
