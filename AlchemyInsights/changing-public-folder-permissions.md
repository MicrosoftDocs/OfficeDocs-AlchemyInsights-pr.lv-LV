# <a name="changing-public-folder-permissions"></a>Mainīšanas publiskās mapes

Publisko mapju atļaujas programmā Outlook varat mainīt jums un jūsu lietotājiem. Sk. [iestatītu vai mainītu publiskas mapes atļaujas](https://support.office.com/article/set-or-change-permissions-for-a-public-folder-b2e0440c-7873-48ec-9ff2-b1a20b723005). Administratori var arī mainīt atļaujas Exchange administratoru centrā:
  
1.  Microsoft 365 admin Center, dodieties uz **Admin centri** \> **apmaiņu**.
    
2. **Publisko mapju**atlasīšana.
    
Šeit varat mainīt atsevišķas publiskās mapes atļaujas, piešķirot drošības grupas lietotājiem atļaujas. Lai mainītu publiskas mapes atļaujas, jums ir jābūt par to īpašnieku.
