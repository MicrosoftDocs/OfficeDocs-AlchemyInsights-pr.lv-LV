# <a name="control-sync-throughput"></a>Vadības sinhronizācijas caurlaidstarpputs

Ja vēlaties kontrolēt sinhronizācijas programmu trafiku, ieteicams, ja iespējams, izmantot tīkla pakalpojumu kvalitātes (QoS) politikas vai Windows QoS politikas. Informāciju par tīkla iestatījumiem, ko varat padarīt pieejamu lietotājiem, skatiet [rakstā OneDrive sinhronizācijas programmu augšupielādes vai lejupielādes ātruma maiņa.](https://support.office.com/article/71cc69da-2371-4981-8cc8-b4558bdda56e)
