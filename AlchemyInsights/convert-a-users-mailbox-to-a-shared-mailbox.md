# <a name="convert-a-user-mailbox-to-a-shared-mailbox"></a>Pārvērstu koplietojamās pastkastes lietotāja pastkastei

Konvertējot Office 365 individuāla lietotāja pastkastes koplietojamās pastkastes, visu esošo e-pasta un kalendāra vienumus izmaiòas koplietojamo pastkastē, kur var piekļūt vairākiem cilvēkiem. Koplietota pastkaste var arī konvertēt atpakaļ uz lietotāju (privātā) pastkastes.
  
Plašāku informāciju skatiet [pārvērst koplietojamās pastkastes lietotāja pastkastes](https://docs.microsoft.com/office365/admin/email/convert-user-mailbox-to-shared-mailbox).
