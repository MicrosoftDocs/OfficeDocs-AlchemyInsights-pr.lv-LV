# <a name="convert-a-user-mailbox-to-a-shared-mailbox"></a>Pārvērstu koplietojamās pastkastes lietotāja pastkastei

Konvertējot koplietojamās pastkastes individuāla lietotāja pastkasti, visu esošo e-pasta un kalendāra vienumus izmaiòas koplietojamo pastkastē, kur var piekļūt vairākiem cilvēkiem. Koplietota pastkaste var arī konvertēt atpakaļ uz lietotāju (privātā) pastkastes.
