# <a name="create-a-site"></a>Izveidot vietni

Var izveidot dažādu tipu vietnēs SharePoint Online. Lai uzzinātu, kā to izdarīt jaunu SharePoint administrēšanas centrā, skatiet [izveidot vietni](https://go.microsoft.com/fwlink/?linkid=866295). Informācija par to, kas ļauj lietotājiem izveidot savas vietnes, skatiet [pārvaldīt vietņu izveide SharePoint Online](https://go.microsoft.com/fwlink/?linkid=866296).
 
Info par jauno vietu tipiem, skatiet [grupas pievienots komandu vietnes](https://go.microsoft.com/fwlink/?linkid=866292) vai [Saziņas vietnes](https://go.microsoft.com/fwlink/?linkid=866294).
