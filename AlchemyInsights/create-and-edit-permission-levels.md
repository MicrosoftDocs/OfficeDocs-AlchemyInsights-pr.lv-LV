# <a name="working-with-sharepoint-permission-levels"></a>Darbs ar SharePoint atļauju līmeņiem

SharePoint komplektācijā ietilpst noklusējuma atļauju līmeņi, kurus var izmantot, piemēram, **Contribute** un **Skatīt**. Taču, ja kāds no šiem neatbilst jūsu vajadzībām, varat izveidot jaunu atļauju līmeni. Ņemiet vērā, ka, ja nemaināt noklusējuma iestatījumus, vieglāk ir sekot atļauju līmeņiem.

[, Kā izveidot un rediģēt atļauju līmeņus](https://docs.microsoft.com/sharepoint/how-to-create-and-edit-permission-levels)
