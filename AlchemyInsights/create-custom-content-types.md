# <a name="create-custom-content-types"></a>Izveidot pielāgotus satura tipus

1. Atlasiet sarakstu, ar kuru vēlaties strādāt ātrās palaišanas joslā, vai noklikšķiniet uz **Iestatījumi** (zobrata ikona) un pēc tam noklikšķiniet uz **vietnes iestatījumi**.
2. Sadaļā **Web Designer galerijas**noklikšķiniet uz **vietnes satura** tipi.
3. Lapā vietnes satura tipi noklikšķiniet uz izveidot.
4. Lapā jauns vietnes satura tips aizpildiet informāciju par jūsu pielāgoto satura tipu.

Papildinformāciju skatiet sadaļā [pielāgoti satura tipi](https://support.microsoft.com/office/e1277a2e-a1e8-4473-9126-91a0647766e5#__toc323548991).
