# <a name="creating-inbox-rules-for-shared-mailboxes"></a>Iesūtnes kārtulu izveide koplietojamām pastkastēm

Koplietojamā pastkastē varat pievienot kārtulas, kurām varat piekļūt līdzīgā veidā, kā varat pievienot kārtulas savam kontam.
  
Pārliecinieties, vai esat attiecīgās koplietojamās pastkastes dalībnieks, un pēc tam izpildiet šajā rakstā norādītās darbības: [kārtulu pievienošana koplietojamai pastkastei](https://support.office.com/article/b0963400-2a51-4c64-afc7-b816d737d164)
