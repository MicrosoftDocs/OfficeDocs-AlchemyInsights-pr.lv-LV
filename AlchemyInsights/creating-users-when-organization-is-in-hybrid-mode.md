# <a name="create-users-in-hybrid-deployments"></a>Lietotāju izveide hibrīdizvietojumos

Kad jūsu organizācija ir Hibrīdā izvietošana lokālā vidē un Microsoft 365, ir jāizveido visi lietotāja objekti lokālā Active Directory. Jūs izveidojat lietotāja objekti atbilstošo organizatoriskajai vienībai (OU), lai tie tiek sinhronizēti jūsu organizācijā.
