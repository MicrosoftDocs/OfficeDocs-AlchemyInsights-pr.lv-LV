# <a name="create-users-in-hybrid-deployments"></a>Izveidot lietotājus hibrīda izvietošana

Ja jūsu organizācija ir hibrīds izvietošanas lokālajā vidē un biroja 365, visi lietotāja objektiem jāizveido jūsu lokālā Active Directory. Veidojat lietotāja objektiem atbilstošu struktūrvienībā (OU), tāpēc tie esam sinhronizēti savā uzņēmumā Office 365.
