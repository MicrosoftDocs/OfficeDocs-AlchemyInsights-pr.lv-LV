# <a name="delegated-mailbox-permissions-in-a-hybrid-environment"></a>Pilnvarotās pastkastes atļaujas hibrīdajā vidē

Atkarībā no jūsu lokālajā organizācijā instalētas Exchange versijas, iespējams, būs jāveic dažas papildu darbības, lai iestatītu pilnvaroto pastkastu atļaujas starp jūsu lokālo organizāciju un Exchange Online pakalpojumā Microsoft 365. Lai iegūtu papildinformāciju par to, kā iestatīt pilnvarotās atļaujas, skatiet rakstu [Exchange konfigurēšana, lai nodrošinātu pilnvaroto pastkastu atļaujas hibrīdajā izvietojumā](https://technet.microsoft.com/library/mt784505%28v=exchg.150%29.aspx).
