# <a name="delegated-mailbox-permissions-in-a-hybrid-environment"></a>Pilnvarotais pastkastes atļaujas hibrīds vidē

Atkarībā no Exchange versija ir instalēta lokālajā organizācijā, iespējams, jāveic dažas papildu darbības, lai iestatītu pilnvarotais pastkastes atļaujas lokālas organizācijas un Exchange Online Microsoft 365. Lai iegūtu papildinformāciju par to, kā iestatīt deleģētās atļaujas, ieskatieties [konfigurēt Exchange atbalsta pilnvarotais pastkastes atļaujas Hibrīdā izvietošana](https://technet.microsoft.com/library/mt784505%28v=exchg.150%29.aspx).
