# <a name="delete-items-or-empty-the-recycle-bin-of-a-sharepoint-site"></a>Vienumu dzēšana vai tukšas vietas SharePoint vietnes atkritne 

Dzēšot vienumu no SharePoint dokumentu bibliotēkas vai saraksta, tas nav uzreiz noņemta no SharePoint. Izdzēstie vienumi uz kādu laika periodu nonāk vietnes atkritnē. Šajā laikā izdzēstos vienumus var atjaunot to sākotnējā atrašanās vietā vai noņemt, lai atbrīvotu vietu krātuvē.
