# <a name="delete-a-sharepoint-site-collection"></a>SharePoint vietņu kolekcijas dzēšanai

1. Klasisks SharePoint administrēšanas centrā, lapā vietņu kolekcijas, atzīmējiet izvēles rūtiņu blakus vietņu kolekcijā.
    
2. Par vietņu kolekcijas cilni, izvēlieties **Dzēst**.
