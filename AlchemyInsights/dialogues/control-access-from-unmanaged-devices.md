# <a name="control-the-access-from-unmanaged-devices"></a>Piekļuves kontrolēšana no nepārvaldītām ierīcēm

Kā SharePoint vai globālais administrators pakalpojumā Microsoft 365 varat bloķēt vai ierobežot piekļuvi SharePoint un OneDrive saturam no nepārvaldītām ierīcēm. Papildinformāciju skatiet rakstā [piekļuves kontrole no nepārvaldītām ierīcēm](https://docs.microsoft.com/sharepoint/control-access-from-unmanaged-devices).
