# <a name="view-or-modify-a-journal-rule"></a>Žurnāla kārtulas skatīšana vai modificēšana

Tālāk aprakstīts, kā to paveikt.

1. [Exchange administrēšanas centrā](https://go.microsoft.com/fwlink/p/?linkid=2059104)dodieties uz sadaļu **atbilstības pārvaldības**  >  **žurnāla kārtulas**.
2. Saraksta skatā redzēsit visas savas organizācijas žurnāla kārtulas.
3. Veiciet dubultklikšķi uz kārtulas, kuru vēlaties skatīt vai modificēt.
4. Sadaļā **Dienasgrāmata** mainiet vēlamos iestatījumus.
