# <a name="disable-comments-on-lists"></a>Komentāru atspējošana sarakstos

Administratori var atspējot šo līdzekli organizācijas līmenī, mainot **CommentsOnListItemsDisabled** parametru **Set-SPOTenant** PowerShell cmdlet.

Pašlaik nav iespējams atspējot komentēšanu vietnes vai saraksta līmenī. Mēs ceram, ka šīs vadīklas tiks atjauninātas vēlāk, iespējams, 2021 pirmajā ceturksnī.
