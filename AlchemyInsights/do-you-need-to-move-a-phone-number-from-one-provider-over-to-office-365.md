# <a name="move-a-phone-number-from-another-provider-to-microsoft"></a>Tālruņa numura pārvietošana no cita pakalpojuma sniedzēja uz Microsoft

Vai jums ir jāpārvieto tālruņa numurs no cita nodrošinātāja uz Microsoft? Dažādu valstu un reģionu tālruņu numurus var atrast [pārvaldīt organizācijas tālruņu numurus](https://docs.microsoft.com/SkypeForBusiness/what-are-calling-plans-in-office-365/manage-phone-numbers-for-your-organization/manage-phone-numbers-for-your-organization). Izvēlieties valsti vai reģionu, lai atrastu informāciju par numuru pārsūtīšanu.
