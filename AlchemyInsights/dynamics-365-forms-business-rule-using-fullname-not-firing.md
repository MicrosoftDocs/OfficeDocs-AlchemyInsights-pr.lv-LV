# <a name="full-name-field-not-supported-with-unified-inteface-apps"></a>Pilns nosaukums lauks netiek atbalstīts ar vienotu inteface progr

Darbības vai apstākļus, kas izmanto lauku **Pilns vārds** (fullname) netiek atbalstītas programmās, pamatojoties uz vienotu interfeisu. Var izmantot arī darbības vai apstākļus ar **vārdu** (vārds) un **Uzvārda** (uzvārds) laukos.
