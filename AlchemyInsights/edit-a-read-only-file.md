# <a name="edit-a-read-only-file"></a>Tikai lasāma faila rediģēšana

Dokumentu bibliotēkās, noklikšķinot uz dokumenta nosaukuma, Office programma varētu negaidīti to atvērt kā tikai lasāmu failu. Lai to atvērtu rediģēšanai, noklikšķiniet uz **Atvērt** (trīs punkti) un pēc tam uz **rediģētu**
