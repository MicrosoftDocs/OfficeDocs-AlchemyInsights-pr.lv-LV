# <a name="edit-a-read-only-file"></a>Tikai lasāma faila rediģēšana

Dokumentu bibliotēkās, kad noklikšķināt uz dokumenta nosaukuma, Office programma, iespējams, to negaidīti atvērs kā tikai lasāmu failu. Lai to atvērtu rediģēšanai, noklikšķiniet uz **Atvērt** (trīs punkti) un pēc tam noklikšķiniet uz **Rediģēt.**
