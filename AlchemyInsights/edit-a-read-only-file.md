# <a name="edit-a-read-only-file"></a>Tikai lasāma faila rediģēšana

Dokumentu bibliotēkās, noklikšķinot uz dokumenta nosaukuma, Office programma negaidīti var atvērt kā tikai lasāmu failu. Lai to atvērtu rediģēšanai, noklikšķiniet uz **Atvērt** (trīs punkti) un pēc tam noklikšķiniet uz **Rediģēt.**
