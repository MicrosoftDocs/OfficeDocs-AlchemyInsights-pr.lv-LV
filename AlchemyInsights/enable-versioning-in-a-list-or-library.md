# <a name="enable-versioning-for-a-list-or-library"></a>Saraksta vai bibliotēkas versiju iespējošana

1. Dodieties uz sarakstu vai bibliotēku.
    
2. Noklikšķiniet uz **Iestatījumi**un pēc tam uz **bibliotēkas iestatījumi** vai **saraksta iestatījumi**.
    
3. Noklikšķiniet uz **versiju iestatījumi**.
    
4. Atlasiet vēlamo iestatījumu un pēc tam noklikšķiniet uz **Labi**.
