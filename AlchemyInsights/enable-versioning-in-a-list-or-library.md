# <a name="enable-versioning-for-a-list-or-library"></a>Iespējot versiju izveidi, sarakstam vai bibliotēkai

1. Dodieties uz sarakstu vai bibliotēku.
    
2. Noklikšķiniet uz **Iestatījumi**un pēc tam noklikšķiniet uz **bibliotēkas** vai **saraksta iestatījumi**.
    
3. Noklikšķiniet uz **versiju iestatījumi**.
    
4. Atlasiet vajadzīgos iestatījumus un pēc tam noklikšķiniet uz **Labi**.
