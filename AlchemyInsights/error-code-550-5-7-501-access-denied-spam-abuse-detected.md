# <a name="550-57501-access-denied-spam-abuse-detected"></a>550 5.7.501 piekļuve liegta, noteikta surogātpasta ļaunprātīga izmantošana

Parasti šis ziņojums tiek parādīts, kad lietotāji sūta e-pasta ziņojumus no IP adresēm, izmantojot sākotnējo *. onmicrosoft.com* domēnu, kas piešķirts jaunajiem nomniekiem pakalpojumā Microsoft 365. Vienkāršākais veids, kā atrisināt šo problēmu, ir:

1. [Domēna pievienošana nomniekam](https://docs.microsoft.com/microsoft-365/admin/setup/add-domain).

2. [Mainiet savu lietotāju primāro e-pasta adresi](https://docs.microsoft.com/microsoft-365/admin/add-users/change-a-user-name-and-email-address) uz jauno pielāgoto domēnu, kuru tikko pievienojāt.
