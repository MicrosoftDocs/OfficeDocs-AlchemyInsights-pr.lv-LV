# <a name="export-user-account-information"></a>Lietotāja konta informācijas eksportēšana

Lai eksportētu lietotāja konta informāciju Microsoft 365 administrēšanas centrā, tostarp informāciju par licencēm, paroļu politiku, bloķētajiem kontiem, AAD sinhronizācijas statusu un citām, pārejiet uz lapu **lietotāju** > [Aktīvie lietotāji](https://go.microsoft.com/fwlink/p/?linkid=834822) un pēc tam atlasiet **Eksportēt lietotājus**.
