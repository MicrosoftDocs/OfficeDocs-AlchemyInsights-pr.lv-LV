# <a name="find-your-pop-or-imap-server-settings"></a>Atrast jūsu POP vai IMAP servera iestatījumus

Ja jūs nezināt, kādi iestatījumi, lai POP vai IMAP konta pievienošana programmai Microsoft Outlook, var meklēt tos ar mūsu [POP un IMAP e-pasta iestatījumus programmā Outlook](https://support.office.com/article/8361e398-8af4-4e97-b147-6c6c4ac95353.aspx).
  
> [!NOTE]
> Vai jūs zināt, jūs varat [Iespējot un konfigurēt POP3 serverī Exchange 2016](https://technet.microsoft.com/library/bb124934%28v=exchg.160%29.aspx)?
