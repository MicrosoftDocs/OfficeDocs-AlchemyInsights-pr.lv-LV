# <a name="new-feature-get-feedback-about-spam-judgments"></a>Jauna iezīme: iegūt atgriezenisko saiti par surogātpasta spriedumi

Kā administrators, varat tagad viegli ziņot aizdomīgu e-pasta ziņojumus, URL un saturu Microsoft [Internet Explorer iesniegumi](https://protection.office.com/reportsubmission). Jūs pat varat izmantot to, lai palīdzētu izprast, kāpēc likumīgu ziņojums bija atzīmēts nepareizi. Lai iegūtu papildinformāciju, skatiet šo [sludinājumu](https://techcommunity.microsoft.com/t5/Security-Privacy-and-Compliance/Empower-security-teams-to-easily-report-suspicious-emails-amp/ba-p/752622).

Lai tā easer, varat atļaut lietotājiem ziņot Microsoft, izmantojot [pievienojumprogrammu atskaites ziņojumu](https://appsource.microsoft.com/product/office/WA104381180?src=office&tab=Overview).
