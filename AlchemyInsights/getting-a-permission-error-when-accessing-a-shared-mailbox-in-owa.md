# <a name="getting-a-permission-error-when-opening-a-shared-mailbox-in-outlook-on-the-web"></a>Atverot koplietojamu pastkastes Outlook Web kļūst atļaujas kļūda?

Ja saņemat kļūdas **kaut kas nogāja greizi. Jums nav atļaujas atvērt šo pastkasti.** mēģinot atvērt koplietojamās pastkastes Outlook tīmeklī, jūsu kontā varētu nav atļaujas piekļūt koplietošanas pastkastes. Jūsu uzņēmuma administrators var piešķirt jūsu konta atļaujas koplietojamās pastkastes, sekojot instrukcijām tēmā [koplietojamo pastkastes izveide](https://support.office.com/article/871a246d-3acd-4bba-948e-5de8be0544c9).
