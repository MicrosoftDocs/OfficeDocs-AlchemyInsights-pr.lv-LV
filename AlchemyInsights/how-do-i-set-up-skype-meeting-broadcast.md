# <a name="cant-start-or-join-a-skype-meeting-broadcast"></a>Nevar startēt vai pievienoties sapulcei Skype apraide?

Ja rodas problēmas, sākot vai pievienoties Skype sapulces apraide kā notikumu grupas dalībnieks (sapulces ražotājs), pārliecinieties, vai vēlreiz pārbaudiet konfigurāciju. Skype sapulces apraidei ir nepieciešamas manuālas iestatīšanas darbības. Izpildiet šajā rakstā aprakstītās darbības, lai nodrošinātu, ka jūsu tīkla un Skype tiešsaistes biznesa portāls ir konfigurēti pareizi un attiecīgā lietotāja nav bloķējis konfliktējošas licences:

[Tīkla iestatīšana Skype sapulču apraidei](https://docs.microsoft.com/SkypeForBusiness/set-up-your-network-for-skype-meeting-broadcast/set-up-your-network-for-skype-meeting-broadcast)
