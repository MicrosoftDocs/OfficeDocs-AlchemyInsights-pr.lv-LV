# <a name="how-to-change-your-theme"></a>Kā mainīt savu dizainu

Mainīt savu dizainu Outlook.com vai programmā Outlook Web, dodieties uz **Iestatījumi** un izvēlieties dizainu, zem opcijas **Tēma**. Lai apskatītu visus pieejamos dizainus, atlasiet **Skatīt visu** saskaņā ar **tēmu**.
