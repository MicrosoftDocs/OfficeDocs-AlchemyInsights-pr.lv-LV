# <a name="how-to-change-your-theme"></a>Kā nomainīt Theme

Lai mainītu dizainu programmā Outlook.com vai Outlook tīmeklī, dodieties uz **Iestatījumi** un izvēlieties motīvu sadaļā **dizains**. Lai skatītu visus pieejamos dizainus, atlasiet **Skatīt visus** sadaļā **dizains**.
