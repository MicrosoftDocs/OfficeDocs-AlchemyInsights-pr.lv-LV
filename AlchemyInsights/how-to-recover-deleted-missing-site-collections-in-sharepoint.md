# <a name="recover-missing-or-deleted-site-collections"></a>Trūkstošo vai izdzēsto vietņu kolekciju atkopšana

Informāciju par to, kā atkopt pazudušas vai izdzēstas vietņu kolekcijas koplietošanas vidē SharePoint, skatiet tālāk norādītajos rakstos.

[Dzēsto saišu atjaunošana](https://docs.microsoft.com/sharepoint/restore-deleted-site-collection)

[Atjaunošana, izmantojot SPO PowerShell moduli](https://support.office.com/article/Introduction-to-the-SharePoint-Online-Management-Shell-C16941C3-19B4-4710-8056-34C034493429)

Ja nevarat atrast vietni, iespējams, vēlēsities [Meklēt audita žurnālu Microsoft 365 drošības &amp; atbilstības centrā](https://docs.microsoft.com/microsoft-365/compliance/search-the-audit-log-in-security-and-compliance).

**Piezīme:** "Modernās" grupas vietnes nav redzamas **klasiskajā** SharePoint administratora UI. Jūs varat piekļūt "modernu" vietņu sarakstam no [jaunā SharePoint administrēšanas centra](https://docs.microsoft.com/sharepoint/get-started-new-admin-center).
