# <a name="identifiers-entity-id"></a>Identifikatori (entītijas ID)

Lai skatītu vai rediģētu SAML pilnvarā emitētās prasības, atveriet lietojumprogrammu Azure portālā. Pēc tam atveriet sadaļu **lietotāja atribūti & prasības** . Lai iegūtu papildinformāciju, skatiet rakstu [SAML pilnvarā izsniegto prasību pielāgošana uzņēmuma lietojumprogrammām](https://docs.microsoft.com/azure/active-directory/develop/active-directory-saml-claims-customization#editing-nameid).
