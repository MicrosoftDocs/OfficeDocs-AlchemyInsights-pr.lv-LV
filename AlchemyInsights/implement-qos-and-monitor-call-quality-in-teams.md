# <a name="implement-qos-and-monitor-call-quality-in-teams"></a>Ieviesiet QoS un uzraugiet zvanu kvalitāti grupās

Zvanu Analytics un zvanu kvalitātes informācijas paneli var izmantot, lai palīdzētu novērst zvanu kvalitātes problēmas Microsoft Teams. Lai uzzinātu vairāk par pakalpojuma kvalitāti (QoS), zvanu analīzi un zvanu kvalitātes informācijas paneli (CQD), skatiet sadaļu [QoS ieviešana un monitora zvanu kvalitāte Microsoft Teams](https://docs.microsoft.com/microsoftteams/monitor-call-quality-qos).
