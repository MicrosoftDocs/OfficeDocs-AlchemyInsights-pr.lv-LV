# <a name="increase-the-size-of-a-shared-mailbox"></a>Palielinātu koplietojamās pastkastes lielumu

Koplietojamās pastkastes lielumu var palielināt no noklusējuma lieluma 50 GB līdz 100 GB, piešķirot koplietojamās pastkastes Exchange Online P2 licences (kas ir iekļauta E3 un E5 licences). Piešķirot P2 licences koplietojamās pastkastes [aktīvo lietotāju](https://portal.office.com/adminportal/home), lielums tiks automātiski palielināts 15 minūšu laikā.
