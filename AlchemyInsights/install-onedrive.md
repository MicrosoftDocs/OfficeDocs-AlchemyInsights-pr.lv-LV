# <a name="install-onedrive"></a>Instalējiet OneDrive

OneDrive ācijas klients ir preinstalēta operētājsistēma Windows 10. Instalēt OneDrive Windows 7 vai Mac, vai lai iegūtu mobilo app tālruni vai tablet, dodieties uz [http://onedrive.com/download](http://onedrive.com/download).
