# <a name="install-project-professional"></a>Instalēt Project Professional

Skatiet sadaļu [projekta instalēšana](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281) , lai uzzinātu, kā instalēt Project Online abonementa versiju un projekta 2019, 2016 vai 2013 versiju, kas nav abonementa versija. 

Project 2010 vai 2007, izpildiet darbības [Office 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d) vai [Office 2007](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657). 

Projekts tiek pārdots kā savrupa lietojumprogramma, tādēļ, ja jums to vēl nav, [iegādājieties vai izmēģiniet projektu](https://products.office.com/project).
