# <a name="join-a-team-by-code"></a>Pievienošanās grupai pēc koda

Microsoft ir informēta par problēmu, kad lietotāji nevar izveidot **savienojumu ar grupas** kodu, vai pievienoties grupai, izmantojot līdzekli **pievienoties pēc koda** . Kā risinājums, grupas īpašnieki var manuāli pievienot lietotājus pēc nepieciešamības.
