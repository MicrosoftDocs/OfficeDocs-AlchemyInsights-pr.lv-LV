# <a name="learn-about-bing-maps-licensing"></a>Informācija par Bing karšu licencēšanu

Bing karšu pakalpojums piedāvā vairākas elastīgas licencēšanas opcijas, kas atbalstīs jūsu organizācijas vajadzības un tehniskās prasības. Turklāt ar bezmaksas līmeni varat sākt veidošanu ar Bing kartēm jau šodien.

Detalizētu informāciju skatiet rakstā [Bing karšu licencēšana](https://go.microsoft.com/fwlink/?linkid=2150203).
