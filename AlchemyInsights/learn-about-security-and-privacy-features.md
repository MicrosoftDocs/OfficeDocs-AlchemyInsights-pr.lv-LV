# <a name="learn-about-the-security-and-privacy-features-of-microsoft-search-in-bing"></a>Informācija par Microsoft Search drošības un konfidencialitātes līdzekļiem pakalpojumā Bing

Microsoft meklēšanas drošības un konfidencialitātes līdzekļi pakalpojumā Bing palīdz organizācijai aizsargāt lietotāju datus un darbavietas datus. Piemēram, tiek veikti pieprasījumi par HTTPS, un savienojumi ir šifrēti no beigām līdz galam.

Papildinformāciju skatiet rakstā [Microsoft Search drošība un konfidencialitāte pakalpojumā Bing](https://docs.microsoft.com/microsoftsearch/security-for-search).
