# <a name="limit-sharing-in-microsoft-365"></a>Koplietošanas ierobežošana programmā Microsoft 365

Lai gan jūs nevarat pilnībā atspējot iekšējo koplietošanu vai noņemt kopīgošanas pogu no vietnēm, ir vairāki veidi, kā varat ierobežot koplietošanu pakalpojumā Microsoft 365, lai apmierinātu savas organizācijas vajadzības. Papildinformāciju skatiet rakstā [kopīgošanas ierobežošana programmā Microsoft 365](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing).
