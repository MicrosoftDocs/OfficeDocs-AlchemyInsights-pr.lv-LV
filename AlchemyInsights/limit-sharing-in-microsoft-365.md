# <a name="limit-sharing-in-microsoft-365"></a>Ierobežot koplietošanu Microsoft 365

Lai gan nevar pilnībā atspējot iekšējo koplietošanu vai noņemt vietņu kopīgošanas pogu, pastāv dažādi veidi, kā varat ierobežot koplietošanu programmā Microsoft 365, lai apmierinātu jūsu organizācijas vajadzības. Lai iegūtu papildinformāciju, skatiet [ierobežot koplietošanu programmā Microsoft 365](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing).
