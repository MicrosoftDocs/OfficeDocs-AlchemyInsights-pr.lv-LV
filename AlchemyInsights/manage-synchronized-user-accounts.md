# <a name="manage-synchronized-user-accounts"></a>Sinhronizētu lietotāju kontu pārvaldība

Parasti jūs izmantojat Microsoft 365 administrēšanas centru, lai pievienotu un mainītu lietotāju kontus. Taču to nevar izmantot, ja ir iespējoti direktorija sinhronizācijas vai SSO. Tā vietā pārvaldiet lietotājus, izmantojot programmu Windows PowerShell, kā aprakstīts rakstā [lietotāja konta rekvizītu konfigurēšana, izmantojot Office 365 PowerShell](https://docs.microsoft.com/office365/enterprise/powershell/configure-user-account-properties-with-office-365-powershell ).
