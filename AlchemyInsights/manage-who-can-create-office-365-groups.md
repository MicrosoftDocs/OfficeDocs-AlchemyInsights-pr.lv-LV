# <a name="manage-who-can-create-microsoft-365-groups"></a>Pārvaldīt, kas var izveidot Microsoft 365 grupas

Pēc noklusējuma ikviens uzņēmuma lietotājs var izveidot O365 grupas.
  
Ja vēlaties mainīt, varat izveidot drošības grupu ar PowerShell, kas ierobežo grupas izveide lietotāju apakškopu.
  
Izpildiet soli-pa-solim instrukcijas šeit: [pārvaldīt kas var izveidot Microsoft 365 grupas](https://docs.microsoft.com/office365/admin/create-groups/manage-creation-of-groups)
