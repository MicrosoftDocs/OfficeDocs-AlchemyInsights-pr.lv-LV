# <a name="map-a-sharepoint-library-to-a-network-drive"></a>Koplietošanas vides SharePoint bibliotēkā Kartēt tīkla disku

Bibliotēka kā tīkla diska kartēšana ir pagaidu un atbalstīja tikai ar Internet Explorer. Jums laikam programmā Internet Explorer atveriet SharePoint vietnes un atlasiet "Ilgāku laiku", lai neļautu sesija beidzas. Tā vietā [SharePoint failu ar jaunu OneDrive sinhronizācijas klienta sinhronizēšana](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88) </a> kas sniedz [Failu On-Demand](https://support.office.com/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e). Piekļūt visiem jūsu failiem OneDrive neizmantojot vietējās uzglabāšanas vietu.

Ja vēlaties kartēt disku, tā vietā, lai[izmantotu jauno OneDrive sinhronizācijas klienta](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88), nodrošina jums sekot zemāk rakstā norādītās darbības. 


**Kā konfigurēt un novērst Kartēt tīkla disku**


- [Konfigurēt un novērst Kartēt tīkla disku](https://support.office.com/article/troubleshoot-mapped-network-drives-that-connect-to-sharepoint-online-ef399c67-4578-4c3a-adbe-0b489084eabe?ui=en-US&amp;rs=en-US&amp;ad=US).

Piezīme: Internet Explorer 10 ar Windows 8, Windows 7, kas saņem "Piekļuve liegta" vai "Ceļš nav pieejams" Kad diska kartēšanu, instalēt šo labojumfailu, lai atrisinātu šo problēmu. [Kļūda, atverot Windows Explorer ar SharePoint dokumentu bibliotēku vai Kartēt tīkla disku uz bibliotēku pēc tam, kad jūs instalējiet Internet Explorer 10](https://support.microsoft.com/help/2846960)nokļūtu.
