# <a name="microphone-isnt-working-in-teams"></a>Mikrofons nedarbojas komandās

Sāciet problēmu novēršana, veicot tālāk norādītās darbības.

- Windows un Teams atjauninājumu meklēšana.
- Aizveriet visas pārējās programmas, kas izmanto mikrofonu.
- Veicot pārbaudes zvanu, pārbaudiet, vai mikrofons darbojas pareizi.

Detalizētu informāciju un citas problēmu novēršanas darbības skatiet rakstā [mans mikrofons nedarbojas pakalpojumā Teams](https://support.microsoft.com/office/666d1123-9dd0-4a31-ad2e-a758b204f33a).
