# <a name="microsoft-365-apps-error-the-application-was-unable-to-start-correctly-0xc0000142"></a>Microsoft 365 programmu kļūda: lietojumprogrammu nevarēja pareizi startēt (0xc0000142)

Lai novērstu šo kļūdu, izmēģiniet tālāk norādītās opcijas.

- Veiciet Office tiešsaistes labošanu un restartējiet datoru. Papildinformāciju skatiet rakstā [Office lietojumprogrammas labošana](https://support.microsoft.com/office/repair-an-office-application-7821d4b6-7c1d-4205-aa0e-a6b40c5bb88b).
- Dodieties uz **Sākt**   >   **izpildi**un pēc tam ierakstiet **Services. msc**. Ar peles labo pogu noklikšķiniet uz  **Microsoft Office Click-to-Run pakalpojumu**un pēc tam atlasiet **Restartēt**.
- Labojiet trūkstošus vai bojātus sistēmas failus, izmantojot sistēmas failu pārbaudītāju, un restartējiet datoru. Papildinformāciju skatiet rakstā [sistēmas failu pārbaudes rīka izmantošana, lai labotu trūkstošus vai bojātus sistēmas failus](https://support.microsoft.com/help/929833/use-the-system-file-checker-tool-to-repair-missing-or-corrupted-system).
- Atinstalējiet un atkārtoti instalējiet Office. Papildinformāciju skatiet rakstā [Office atinstalēšana personālajā datorā](https://support.microsoft.com/office/uninstall-office-from-a-pc-9dd49b83-264a-477a-8fcc-2fdf5dbf61d8).
