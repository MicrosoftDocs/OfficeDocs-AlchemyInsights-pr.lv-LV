# <a name="microsoft-365-guest-sharing-settings-reference"></a>Microsoft 365 kopīgošanas ar viesiem iestatījumu atsauce

Šajā rakstā ir sniegta atsauce uz dažādiem iestatījumiem, kas var ietekmēt kopīgošanu ar personām ārpus jūsu organizācijas Microsoft 365 darba slodzēm: Teams, Office 365 grupas, SharePoint un OneDrive. Šie iestatījumi atrodas Azure Active Directory, Microsoft 365, Teams un SharePoint administrēšanas centros.

- [Microsoft 365 kopīgošanas ar viesiem iestatījumu atsauce](https://docs.microsoft.com/microsoft-365/solutions/microsoft-365-guest-settings?view=o365-worldwide)
