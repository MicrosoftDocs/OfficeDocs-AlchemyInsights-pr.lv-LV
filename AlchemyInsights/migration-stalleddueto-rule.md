# <a name="stalleddueto-errors-during-migration"></a>StalledDueTo kļūdas migrācijas laikā

Iespējams, redzēsit dažādas kļūdas, kas sākas ar "StalledDueTo..." migrācijas laikā uz Exchange Online. Daudzas no šīm kļūdām ir pilnīgi parastas. Papildinformāciju skatiet rakstā [resursu ierobežošana un prioritāšu noteikšana pakalpojumā Exchange Online migrations](https://techcommunity.microsoft.com/t5/exchange-team-blog/resource-based-throttling-and-prioritization-in-exchange-online/ba-p/608020).
