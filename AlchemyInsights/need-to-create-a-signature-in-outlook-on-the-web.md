# <a name="need-to-create-an-email-signature"></a>Vai vēlaties izveidot e-pasta parakstu?

Ja jums ir Outlook konts vai Outlook tīmeklī (darba vai mācību iestādes) konts, varat izveidot parakstu un to automātiski pievienot visiem izejošajiem ziņojumiem vai selektīvi pievienot konkrētiem izejošajiem ziņojumiem. Ņemiet vērā, ka jūsu e-pasta paraksts programmā Outlook un Outlook tīmeklī atšķiras. Lai izveidotu savu e-pasta parakstu, skatiet rakstu [Paraksta izveide un pievienošana ziņojumiem programmā Outlook](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx) , kā [arī e-pasta paraksta izveide un pievienošana programmā Outlook tīmeklī](https://support.office.com/article/5ff9dcfd-d3f1-447b-b2e9-39f91b074ea3.aspx).
