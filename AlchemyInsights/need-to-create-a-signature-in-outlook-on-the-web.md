# <a name="need-to-create-an-email-signature"></a>Ir jāizveido e-parakstu?

Ja jums ir konts Outlook vai Outlook web (Office 365 darbā vai skolā) kontā, var izveidot paraksta un automātiski pievienot to visiem izejošajiem ziņojumiem vai selektīvi pievienot īpašas izejošajos ziņojumos. Ņemiet vērā, ka jūsu e-pasta paraksts ir dažādās programmās Outlook un Outlook Web. Lai izveidotu e-parakstu, skatiet [izveidošana un paraksta pievienošana ziņojumiem programmā Outlook](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx) vai [izveidot un pievienot e-pasta parakstu, Outlook Web](https://support.office.com/article/5ff9dcfd-d3f1-447b-b2e9-39f91b074ea3.aspx).
