# <a name="need-to-create-an-email-signature"></a>Nepieciešams izveidot e-pasta parakstu?

Ja jums ir Outlook konts vai Outlook Web (darba vai skolas) konts, varat izveidot parakstu un automātiski pievienot to visiem izejošajiem ziņojumiem vai selektīvi pievienot tos konkrētiem izejošajiem ziņojumiem. Ņemiet vērā, ka jūsu e-pasta paraksts atšķiras programmā Outlook un Outlook tīmeklī. Lai izveidotu savu e-pasta parakstu, skatiet [izveidot un pievienot parakstu ziņojumiem programmā Outlook](https://support.office.com/article/8ee5d4f4-68fd-464a-a1c1-0e1c80bb27f2.aspx) vai [izveidot un pievienot e-pasta parakstu programmā Outlook tīmeklī](https://support.office.com/article/5ff9dcfd-d3f1-447b-b2e9-39f91b074ea3.aspx).
