# <a name="there-isnt-enough-memory-to-complete-this-action"></a>Nepietiek atmiņas, lai pabeigtu šo darbību

Strādājot ar Microsoft Excel failu, tiek parādīts kāds no šiem ziņojumiem:

- Programma Excel nevar pabeigt šo uzdevumu, izmantojot pieejamos resursus. Izvēlieties mazāk datu vai aizveriet citas lietojumprogrammas.
- Nav atmiņas
- Nepietiek sistēmas resursu, lai pilnībā parādītu
- Nepietiek atmiņas, lai pabeigtu šo darbību. Mēģiniet izmantot mazāk datu vai aizvērt citas lietojumprogrammas. Lai palielinātu atmiņas pieejamību, apsveriet: 
    - , Izmantojot Microsoft Excel 64 bitu versiju.
    - Atmiņas pievienošana ierīcei.

Lai novērstu šīs kļūdas, skatiet, [kā novērst "pieejamie resursi" kļūdas programmā Excel](https://docs.microsoft.com/office/troubleshoot/excel/available-resources-errors).
