# <a name="one-of-your-on-premises-federation-service-certificates-is-expiring"></a>Viens no lokālajiem Federācijas pakalpojuma sertifikātiem beidzas

Lai novērstu šo problēmu, rīkojieties šādi:
  
- Instalējiet Microsoft Azure Active Directory modulis Windows PowerShell datorā (ja modulis jau nav instalēts). Lai to izdarītu, dodieties uz [Azure Active Directory PowerShell Graph](https://docs.microsoft.com/powershell/azure/active-directory/install-adv2?view=azureadps-2.0)
    
- Izpildiet darbības "1. scenārijs: AD FS pilnvaru parakstīšanas sertifikāta derīgums" sadaļā ["radās problēma, piekļūstot vietnes" kļūda no ad FS, ārējam lietotājam pierakstās Microsoft 365, Azure vai InTune](https://support.microsoft.com/help/2713898/there-was-a-problem-accessing-the-site-error-from-ad-fs-when-a-federat).
    
- Izpildiet norādījumus, [kā atjaunināt vai labot integrētās domēna iestatījumus Microsoft 365, Azure vai InTune](https://support.microsoft.com/help/2647048/how-to-update-or-repair-the-settings-of-a-federated-domain-in-office-3).
    
Lai iegūtu papildinformāciju par atjaunošanas Federācijas sertifikātu, skatiet [sertifikātu atjaunošanu O365 un AZURE ad](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-o365-certs).
