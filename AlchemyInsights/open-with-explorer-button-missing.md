# <a name="the-open-with-explorer-button-is-missing"></a>Ar Internet Explorer pogu atvērt trūkst

**Atvērt ar Internet Explorer** pogas neparādās mūsdienu pieredze SharePoint un OneDrive. Lai File Explorer atveriet dokumentu bibliotēku, tā programmā Internet Explorer pārlūkojiet un atlasiet izvēlnes skats \> **failu Explorer skats**. **Failu Explorer skats** opcija nav pieejama programmā Microsoft Edge, Google Chrome un Firefox. 
  
> [!NOTE]
> Atvērt, izmantojot Internet Explorer tiek atbalstīta tikai ar Internet Explorer. Tā vietā [SharePoint failu ar jaunu OneDrive sinhronizācijas klienta sinhronizēšana](https://support.office.com/article/6de9ede8-5b6e-4503-80b2-6190f3354a88.aspx) kas sniedz [Failu On-Demand](https://support.office.com/article/0e6860d3-d9f3-4971-b321-7092438fb38e.aspx). Piekļūt visiem jūsu failiem OneDrive neizmantojot vietējās uzglabāšanas vietu.
