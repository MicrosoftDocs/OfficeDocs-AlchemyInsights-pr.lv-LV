# <a name="adblock"></a>Adblock

Daži lietotāji pašlaik rodas problēmas, lasot vai sastādot e-pastu, izmantojot Google Chrome AdBlock paplašinājumu. Lai novērstu šo problēmu:

1. Izmantojiet [Microsoft Edge](https://www.microsoft.com/windows/microsoft-edge) vai citu tīmekļa pārlūkprogrammu.
1. Izslēgt Outlook.com AdBlock iestatījumus.Google Chrome augšējā labajā stūrī atlasiet **Adblock** un pēc tam atlasiet **nepalaist lapās šajā domēnā**.Atsvaidziniet pārlūkprogrammu, lai ielādētu Outlook.com vēlreiz.
