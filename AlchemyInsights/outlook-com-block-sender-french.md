# <a name="bloquer-ou-dbloquer-expditeurs-dans-outlookcom"></a>Bloquer ou débloquer expéditeurs dans Outlook.com

SI vous recevez du courrier indésirable, vous pouvez bloquer les adresses de messagerie et des domaines que vous ne voulez pas recevoir des ziņojumus. Ajouter une adresse de messagerie ou ANO domaine à jūsu liste des expéditeurs bloqués pour déplacer ces ziņojumus directement vers jūsu dosjē courrier indésirable.

1. Accédez à **Paramètres** > **Opcijas**.
2. Dans le volet gauche, sélectionnez **courrier indésirable** > **expéditeurs bloqués**.
3. Entrez l ' Adresse de messagerie dans **les expéditeurs bloqués**, sélectionnez **Ajouter**.
4. Sélectionnez **Enregistrer**.

Ieliet plus d'informations, lisez [Bloquer ou débloquer expéditeurs dans Outlook.com](https://support.office.com/fr-fr/article/bloquer-ou-débloquer-expéditeurs-dans-outlook-com-afba1c94-77bb-4f50-8b85-057cf52f4d5e?ui=fr-FR&rs=fr-FR&ad=FR).
