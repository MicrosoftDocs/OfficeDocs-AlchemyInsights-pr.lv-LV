# <a name="how-to-type-special-characters-on-a-mac"></a>Kā lai rakstītu speciālās rakstzīmes, Mac

Lai rakstītu speciālās rakstzīmes jūsu Mac, atveriet izvēlni **speciālās rakstzīmes** ar tastatūras īsceļu ⌘ **komandu** + **kontroles** + **vietas**.

Ātrās saīsni @ ir arī **ALT** + **64**.
