# <a name="how-to-type-special-characters-on-a-mac"></a>Kā ierakstīt speciālās rakstzīmes uz Mac

Lai Mac datorā rakstītu speciālās rakstzīmes, atveriet **speciālo rakstzīmju** izvēlni, nospiežot īsinājumtaustiņu ⌘ **komandas** + **vadības** + **telpa**.

Quick īsceļu @ ir arī **ALT** + **64**.
