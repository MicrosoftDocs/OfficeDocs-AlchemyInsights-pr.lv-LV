# <a name="copy-and-paste-in-outlookcom"></a>Kopēt un ielīmēt Outlook.com

Outlook.com nav kopēšanas vai ielīmēšanas pogas, bet jūs varat izmantot tastatūras īsinājumtaustiņus:

Datorā **:** CTRL + C lai kopētu, CTRL + X samazināt, un CTRL + V ielīmēt.

**Uz Mac:** CMD + C lai kopētu, CMD + X samazināt, un CMD + V ielīmēt.
