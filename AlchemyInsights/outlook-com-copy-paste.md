# <a name="copy-and-paste-in-outlookcom"></a>Nokopējiet un ielīmējiet Outlook.com

Nav kopēt vai Ielīmēt pogas Outlook.com, bet jūs varat izmantot tastatūras īsinājumtaustiņus:

**Uz PC:** CTRL + C, lai kopētu un CTRL + X izgriešanai, CTRL + V, lai ielīmētu.

**Mac:** CMD + C, lai kopētu, CMD + X izgriešanai un CMD + V, lai ielīmētu.
