# <a name="copy-and-paste-in-outlookcom"></a>Kopēšana un ielīmēšana programmā Outlook.com

Programmā Outlook.com netiek rādītas pogas Kopēt un ielīmēt, taču varat izmantot īsinājumtaustiņus.

**Datorā:** CTRL + C, lai kopētu, CTRL + X, lai izgrieztu, un taustiņu kombināciju CTRL + V, lai to ielīmētu.

**Mac datorā:** CMD + C, lai kopētu, CMD + X, lai izgrieztu, un CMD + V, lai ielīmētu.
