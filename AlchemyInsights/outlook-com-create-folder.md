# <a name="create-a-folder"></a>Izveidot mapi

1. Mapju rūts apakšdaļā atlasiet **Jauna mape**. Iespējams, ka vajadzēs ritināt uz leju, lai redzētu **jaunās mapes** pogu mapju rūtī.
2. Tekstlodziņā ievadiet mapes nosaukumu un pēc tam nospiediet taustiņu Enter.

Lai uzzinātu vairāk, skatiet [izveidot mapi Outlook.com](https://support.office.com/article/6bb0723a-f39f-4a8d-bb3f-fab5dcc2510a?wt.mc_id=Office_Outlook_com_Alchemy).
