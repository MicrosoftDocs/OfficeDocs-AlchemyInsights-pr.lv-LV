# <a name="change-your-pin"></a>Mainīt PIN

Outlook.com neizmanto PIN, bet jūs varat pierakstīties datorā, izmantojot PIN vai Windows Hello. Lai mainītu PIN Windows 10:

1. Atlasiet pogu Sākums vai nospiediet Windows tastatūras taustiņu un pēc tam atlasiet **iestatījumu** > **kontu** > **pierakstīšanās opcijas**.
2. Sadaļā **Piespraust**atlasiet pogu **mainīt** .
3. Ievadiet savu pašreizējo PIN, pēc tam ievadiet un apstipriniet savu jauno PIN zem.

[Lasiet vairāk par Windows Hello.](https://support.microsoft.com/help/17215/)
