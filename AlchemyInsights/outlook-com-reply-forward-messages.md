# <a name="how-to-reply-to-or-forward-messages"></a>Kā uz to atbildēt vai pārsūtīt ziņojumus

Ziņojuma rūts augšējā labajā stūrī, atlasiet pa kreisi, līkā bultiņas, kreisās pārkares dubultbultiņu vai pretējās bultiņas pa labi, lai atbildēt, atbildēt visiem vai pārsūtīt.
  
Uzziniet vairāk par to, kā [izveidot, atbildēt, vai pārsūtīt e-pasta ziņojumi Outlook.com](https://support.office.com/article/5a240eb5-8840-4146-b5e8-b078dce6e5e4?wt.mc_id=Office_Outlook_com_Alchemy).
