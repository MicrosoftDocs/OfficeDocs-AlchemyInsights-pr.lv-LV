# <a name="how-to-select-all-in-outlookcom"></a>Kā "atlasiet visas" Outlook.com

Var atlasīt tikai viss, ja visi ziņojumi ir redzami. Tas nepieciešams, lai novērstu nejaušu darbību ziņojumiem, jūs nevarat redzēt.

**CTRL + klikšķis** atlasa vairākus atsevišķus ziņojumus.

**Taustiņu SHIFT + noklikšķiniet** izvēlas starp pirmo un pēdējo ziņojumu atzīmējat visas ziņas.
