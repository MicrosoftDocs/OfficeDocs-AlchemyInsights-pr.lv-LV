# <a name="how-to-select-all-in-outlookcom"></a>Kā "Select All" Outlook.com

Visus ziņojumus var atlasīt tikai tad, ja ir redzami visi ziņojumi. Tas nepieciešams, lai novērstu nejaušu darbību ziņojumiem, kurus nevarat redzēt.

**CTRL + klikšķis** atlasa vairākus atsevišķus ziņojumus.

**Shift + klikšķis** atlasa visus ziņojumus starp pirmo un pēdējo atlasīto ziņojumu.
