# <a name="how-to-select-all-in-outlookcom"></a>Kā "Atlasīt visu" pievienojumprogrammā Outlook.com

Varat atlasīt tikai visus, ja ir redzami visi ziņojumi. Tas ir, lai novērstu nejaušas darbības ar ziņojumiem, ko neredzat.

**CTRL + klikšķis** atlasa vairākus atsevišķus ziņojumus.

**SHIFT + klikšķis** atlasa visus ziņojumus starp pirmo un pēdējo atlasīto ziņojumu.
