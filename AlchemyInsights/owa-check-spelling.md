# <a name="use-your-browser-to-check-spelling"></a>Izmantojiet savu pārlūkprogrammu, lai pārbaudītu pareizrakstību

Nav pareizrakstības izvēles iespēju programmā Outlook tīmeklī. Lai pārbaudītu pareizrakstību, jums vajadzēs izmantot savu web pārlūkprogrammu. Pareizrakstības pārbaude ir pieejams Microsoft Edge Internet Explorer 10 un jaunākas versijas un pašreizējās versijas Firefox, Chrome, Safari. Pārbaudiet savu web pārlūkprogrammu, lai uzzinātu vairāk par to, kā pārbaudīt pareizrakstību opcijas.
