# <a name="import-contacts"></a>Importēt kontaktpersonas

Pirms importējat Outlook tīmeklī, jums tās būs jāeksportē no jūsu citām e-pasta pakalpojumu sniedzējs kā. csv failu. Izpildiet tālāk minētos norādījumus.

1. Lapas apakšdaļā atlasiet ikonu cilvēki iet cilvēki lapu.
2. Rīkjoslā atlasiet **Pārvaldīt** > **Importēt kontaktpersonas**.
3. Atlasiet **Pārlūkot**, izvēlēties savu CSV failu un pēc tam **Atvērt**.
4. Atlasiet **Importēt**.

Lai uzzinātu vairāk, skatiet [Importēt kontaktpersonas programmā Outlook](https://support.office.com/article/bb796340-b58a-46c1-90c7-b549b8f3c5f8#ID0EAACAAA=Outlook_on_the_web).
