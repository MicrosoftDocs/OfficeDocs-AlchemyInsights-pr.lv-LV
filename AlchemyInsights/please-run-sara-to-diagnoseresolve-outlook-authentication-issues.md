# <a name="use-sara-to-diagnose-and-resolve-outlook-authentication-issues"></a>SaRA jālieto diagnosticē un risina ar Outlook autentifikāciju jautājumiem

Mēs ļoti iesakām izmantojat [Outlook glabā lūdzot manu paroli](https://aka.ms/SaRA-OutlookPwdPrompt-Alchemy) diagnostikas skarto datorā novērst problēmas ja Outlook pastāvīgi pieprasa paroli. Šo [SaRA](https://diagnostics.office.com/#/) diagnostikas vai automatizētas kontroles un atgriež iespējamos risinājumus var izmantot, lai risinātu visas atklātās problēmas.
