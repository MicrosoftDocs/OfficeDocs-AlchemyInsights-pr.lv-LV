# <a name="profile-property-synchronization"></a>Profila rekvizītu sinhronizācija

Nav iespējams kartēt papildu rekvizītus reklāmas importēšanai, lai sinhronizētu no Azure Active Directory uz lietotāja profila programmu. AD Import ir ierobežota uz iepriekš konfigurētu rekvizītu kopu, lai nodrošinātu konsekventu taimera darba veiktspēju.
  
[Papildinformācija par sinhronizācijas profila rekvizītiem](https://go.microsoft.com/fwlink/?linkid=875671)
