# <a name="proxyaddress-incorrect"></a>Nepareiza ProxyAddress

Kad objekts tiek sinhronizēts Azure AD vērtības, kas norādītas atribūtā proxyAddresses Active Directory tiek salīdzinātas ar Azure AD noteikumiem un pēc tam proxyAddresses atribūts tiek aizpildīts Azure AD. Tādēļ proxyAddresses atribūtu objekta Active Directory vērtības nevar būt tāds pats kā proxyAddresses atribūtu Azure AD vērtības.
  
Lai iegūtu papildinformāciju par to, kā tiek aizpildīta proxyaddress, skatiet [kā proxyaddress atribūts tiek aizpildīts AZURE ad](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
