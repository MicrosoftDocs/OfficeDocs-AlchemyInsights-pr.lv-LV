# <a name="public-folder-access-from-mobile-devices"></a>Piekļuve publiskajām mapēm no mobilajām ierīcēm

Piekļuve publiskajām mapēm nedarbojas, izmantojot mobilās ierīces, lūdzu, izmantojiet OWA (Outlook tīmeklī) vai Outlook operētājsistēmai Windows datora vai Outlook darbam ar Mac klientiem, lai piekļūtu publiskajām mapēm.

Ja vēlaties izmantot funkcionalitāti, kas ir līdzīga publiskajām mapēm, un jums ir saturs, kas pieejams mobilajās ierīcēs, skatiet [informāciju par Microsoft 365 grupām](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) , lai iegūtu labāku alternatīvu.
