# <a name="publishing-infrastructure-with-a-communication-site"></a>Publicēšanas infrastruktūru ar saziņas vietne


SharePoint Server publicēšanas infrastruktūru līdzekļa iespējošanas ir ar saziņas vietnes **netiek atbalstīta** . 
  
Lai iegūtu vairāk informācijas, sk. [Moving no izdevējdarbības vietām uz saziņas vietnes](https://docs.microsoft.com/sharepoint/publishing-sites-classic-to-modern-experience).
