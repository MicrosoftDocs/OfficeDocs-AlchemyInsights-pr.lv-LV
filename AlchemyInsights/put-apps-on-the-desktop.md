# <a name="put-apps-on-the-desktop"></a>Programmu iešana darbvirsmā

Meklējiet lietotni **sākuma** izvēlnē un satveriet lietotnes ikonu, turot nospiestu peles kreiso pogu un pārvietojot ikonu uz darbvirsmu.

Var arī nospiest **Windows taustiņu + R** un pēc tam ierakstiet šo komandu:

`shell:appsfolder`

Tas atvērs mapi **Applications** **failu pārlūks**. Tagad jūs varat vilkt un nomest lietotnes ikonu darbvirsmā.
