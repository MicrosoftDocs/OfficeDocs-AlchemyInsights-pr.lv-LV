1. <span data-ttu-id="6ec4b-104">Nekavējoties [atiestatiet lietotāja paroli](https://support.office.com/article/7a5d073b-7fae-4aa5-8f96-9ecd041aba9c).</span><span class="sxs-lookup"><span data-stu-id="6ec4b-104">[Reset the user's password](https://support.office.com/article/7a5d073b-7fae-4aa5-8f96-9ecd041aba9c) immediately.</span></span> <span data-ttu-id="6ec4b-105">Nesūtiet jauno paroli lietotājam pa e-pastu.</span><span class="sxs-lookup"><span data-stu-id="6ec4b-105">Do not communicate the new password through email to the end user.</span></span>

3. <span data-ttu-id="6ec4b-107">Noņemiet visas aizdomīgās [iesūtnes kārtulas](https://support.office.com/article/1433E3A0-7FB0-4999-B536-50E05CB67FED), kas iestatītas pastkastē.</span><span class="sxs-lookup"><span data-stu-id="6ec4b-107">Remove any suspicious [inbox rules](https://support.office.com/article/1433E3A0-7FB0-4999-B536-50E05CB67FED) set within the mailbox.</span></span>

4. <span data-ttu-id="6ec4b-108">Ja lietotājs ir bloķēts, neļaujot tam nosūtīt e-pasta ziņojumu, [dodieties uz sadaļu Lietotāji ar ierobežotām tiesībām, lai atbloķētu kontu](https://protection.office.com/?hash=/restrictedusers).</span><span class="sxs-lookup"><span data-stu-id="6ec4b-108">If the user is blocked from sending email, [go to the Restricted Users to unblock the account](https://protection.office.com/?hash=/restrictedusers).</span></span> <span data-ttu-id="6ec4b-109">Kad tas ir paveikts, lietotājs vienas stundas laikā varēs atsākt sūtīt ziņojumus.</span><span class="sxs-lookup"><span data-stu-id="6ec4b-109">Once done, the user should be able to resume sending messages within 1 hour.</span></span>
