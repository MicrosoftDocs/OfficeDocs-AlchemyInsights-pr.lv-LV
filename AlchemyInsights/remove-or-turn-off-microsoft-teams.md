# <a name="remove-or-turn-off-microsoft-teams"></a>Microsoft Teams noņemšana vai izslēgšana

Vai jūsu uzņēmums vēl nav gatavs [sadarboties vai strādāt attāli, izmantojot Microsoft Teams](https://products.office.com/microsoft-teams/group-chat-software?&OCID=AID2000955_SEM_WiLWtgAAAKcGoHNG:20200305184100:s&msclkid=cbe12a5675e41135662d7437325dbd9a&ef_id=WiLWtgAAAKcGoHNG:20200305184100:s)?

- Kā [atinstalēt Microsoft Teams](https://support.office.com/article/Uninstall-Microsoft-Teams-3b159754-3c26-4952-abe7-57d27f5f4c81).

- [Kā ieslēgt vai izslēgt darba grupas visam uzņēmumam](https://docs.microsoft.com/MicrosoftTeams/office-365-set-up).

- [Neļaut komandām automātiski sākt ar logu 10, noņemot to no startēšanas izvēlnes](https://support.microsoft.com/help/4026268/windows-10-change-startup-apps).

- [Izmantojiet grupas politiku, lai kontrolētu Microsoft Teams instalēšanu](https://docs.microsoft.com/deployoffice/teams-install#use-group-policy-to-control-the-installation-of-microsoft-teams).

- [Izmantojiet grupas politiku, lai neļautu Microsoft komandām startēties automātiski pēc instalēšanas](https://docs.microsoft.com/deployoffice/teams-install#use-group-policy-to-prevent-microsoft-teams-from-starting-automatically-after-installation).

- [Koplietota datora un VDI vidē ar Microsoft Teams](https://docs.microsoft.com/deployoffice/teams-install#shared-computer-and-vdi-environments-with-microsoft-teams).
