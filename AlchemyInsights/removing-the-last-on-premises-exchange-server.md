# <a name="removing-the-last-on-premises-exchange-server"></a>Noņemot pēdējo lokālo Exchange Server

Pirms noņemat pēdējo Exchange serveri no lokālās organizācijas, jums ir jāpārliecinās, ka saprotat, kā to izdarīt, lai palīdzētu jums izlemt, vai jums ir nepieciešams, lai pēdējais serveris jebkādiem citiem mērķiem. Lai iegūtu papildinformāciju, skatiet [kā un kad pārtraukt lokālas Exchange serveri Hibrīdā izvietošana](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).
