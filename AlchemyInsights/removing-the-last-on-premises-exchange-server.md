# <a name="removing-the-last-on-premises-exchange-server"></a>Noņemot pēdējo lokālā Exchange servera

Pirms izņemšanas no jūsu lokālajā uzņēmumā pēdējo Exchange server, jums jāpārliecinās, jums saprast sekas to izdarīt, lai palīdzēt jums izlemt, ja jums šī pēdējā servera citiem nolūkiem. Lai iegūtu papildinformāciju, pārbaudiet [kā un kad pārtraukt ekspluatāciju lokālā Exchange serveru hibrīda izvietošanas](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).
