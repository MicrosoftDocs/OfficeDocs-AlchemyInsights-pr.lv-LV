# <a name="would-you-like-to-report-a-spam-false-positive-to-microsoft"></a>Vai vēlaties ziņot par surogātpasta viltus pozitīvi uz Microsoft?

Būdams administrators, vislabākais veids, kā noteikt, kāpēc tika piegādāts pikšķerēšanas ziņojums, ir iesniegt paraugu korporācijai Microsoft, [pieteikumos](https://protection.office.com/reportsubmission). Lai padarītu to easer, varat atļaut lietotājiem ziņot tieši korporācijai Microsoft, izmantojot [atskaites ziņojuma pievienojumprogrammu](https://appsource.microsoft.com/product/office/WA104381180?src=office&tab=Overview).

Iesakām ievērot brīdinājumu, ja izlemjat [apiet Microsoft 365 surogātpasta filtrus](https://docs.microsoft.com/exchange/troubleshoot/antispam/cautions-against-bypassing-spam-filters).

**Vai zinājāt?** tagad varat iesniegt paraugus tieši no [ziņojumu izsekošanas](https://protection.office.com/messagetrace) un E5/ATP 2. plāns klienti var iesniegt paraugus no [draudu pārlūka](https://docs.microsoft.com/microsoft-365/security/office-365-security/threat-explorer).
