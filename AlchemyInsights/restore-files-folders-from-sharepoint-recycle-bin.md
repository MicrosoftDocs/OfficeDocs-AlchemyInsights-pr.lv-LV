# <a name="restore-files-or-folders-from-the-sharepoint-recycle-bin"></a>Failu vai mapju atjaunošana no SharePoint atkritnes 

Izdzēšot vienumu no SharePoint dokumentu bibliotēkas vai saraksta, tas netiek uzreiz noņemts no SharePoint. Izdzēstie vienumi laika periodā tiek ievietoti vietnes atkritnē. Šajā laikā varat atjaunot vienumus, kurus izdzēsāt to sākotnējā atrašanās vietā.

- [Izdzēsto failu un mapju atjaunošana pakalpojumā OneDrive](https://support.office.com/article/restore-deleted-files-or-folders-in-onedrive-949ada80-0026-4db3-a953-c99083e6a84f)
