# <a name="restore-files-or-folders-from-the-sharepoint-recycle-bin"></a>Failu vai mapju atjaunošana no SharePoint atkritnes 

Dzēšot vienumu no SharePoint dokumentu bibliotēkas vai saraksta, tas nav uzreiz noņemta no SharePoint. Izdzēstie vienumi uz kādu laika periodu nonāk vietnes atkritnē. Šajā laikā izdzēstos vienumus var atjaunot to sākotnējā atrašanās vietā.

- [Failu vai mapju atjaunošana no SharePoint atkritnes](https://support.office.com/article/Restore-items-in-the-Recycle-Bin-of-a-SharePoint-site-6df466b6-55f2-4898-8d6e-c0dff851a0be)
- [Izdzēsto failu vai mapju atjaunošana pakalpojumā OneDrive](https://support.office.com/article/restore-deleted-files-or-folders-in-onedrive-949ada80-0026-4db3-a953-c99083e6a84f)
