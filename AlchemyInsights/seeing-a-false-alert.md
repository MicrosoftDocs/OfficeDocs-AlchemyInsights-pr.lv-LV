- Iesniegt failu Microsoft analīzei. Papildinformāciju skatiet sadaļā Kā [rīkoties ar viltus pozitīviem pozitīviem/negatīviem pretvīrusu programmatūrā Microsoft Defender](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#submit-a-file-to-microsoft-for-analysis).

Papildinformāciju skatiet sadaļā [Izslēgšanas definēšanu atsevišķā Windows ierīcē, lai neļautu skenēt vienumu](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-antivirus/antivirus-false-positives-negatives#define-an-exclusion-on-an-individual-windows-device-to-prevent-an-item-from-being-scanned).
