# <a name="service-principal-key"></a>Pakalpojuma galvenā atslēga

Norādījumus par to, kā izmantot galveno akreditācijas datu pārvaldības pakalpojumu principālam, skatiet šajos rakstos:

- [Get-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/get-azureadserviceprincipalkeycredential) -lai saņemtu galvenos akreditācijas datus pakalpojumam, izmantojot PowerShell.

- [New-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/new-azureadserviceprincipalkeycredential) -izveidot jaunas atslēgas akreditācijas datus pakalpojuma galvenajai lietošanai, izmantojot PowerShell.

- [Noņemt-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/remove-azureadserviceprincipalkeycredential) -dzēst atslēgas akreditācijas datus, izmantojot PowerShell.
