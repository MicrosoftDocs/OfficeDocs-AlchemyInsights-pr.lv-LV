# <a name="set-passwords-to-never-expire"></a>Iestatīt paroles, lai nekad beidzas

Iestatīt paroles, lai nekad beidzas no [drošību &amp; privātuma](https://portal.office.com/adminportal/home#/settings/security). Noklikšķiniet uz "Rediģēt" blakus paroļu politika, un savukārt slīdni, lai "par".
  
Sīkākai informācijai sk.: [iestatīt paroli termiņa politiku uzņēmumā](https://support.office.com/en-us/article/set-the-password-expiration-policy-for-your-organization-0f54736f-eb22-414c-8273-498a0918678f).
