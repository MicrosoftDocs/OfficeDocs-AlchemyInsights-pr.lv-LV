# <a name="setting-up-a-hybrid-environment"></a>Hibrīdas vides iestatīšana

Lai ļautu lietotājiem netraucēti sadarboties neatkarīgi no tā, vai viņu dati tiek glabāti lokālajos serveros vai programmā Microsoft 365, iespējams, ir jāiestata hibrīda vide. Lai iegūtu pārskatu par opcijām, skatiet [Microsoft 365 integrāciju ar lokālajām vidēm](https://docs.microsoft.com/office365/enterprise/office-365-integration).
