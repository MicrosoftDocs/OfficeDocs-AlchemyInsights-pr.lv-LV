# <a name="set-up-sharepoint"></a>SharePoint iestatīšana

[SharePoint iestatīšanas ceļvedis](https://go.microsoft.com/fwlink/?linkid=2071425) palīdz iestatīt SharePoint dokumentu krātuves un satura pārvaldību, izveidot vietnes, konfigurēt ārējo koplietošanu, migrēt datus un konfigurēt papildu iestatījumus, kā arī vadīt lietotāju iesaistīšanos un saziņu jūsu organizācijā.

Izpildiet darbības, lai konfigurētu satura koplietošanas atļauju politikas, izvēlieties migrācijas sinhronizācijas rīkus, kā arī iespējojiet SharePoint vides drošības iestatījumus.
