# <a name="issues-while-migrating-data-to-sharepoint-online"></a>Jautājumus tiešsaistē SharePoint datu migrēšanas laikā

Šķiet, ka jūs migrē virs 100TB datus SharePoint Online.

Lūdzu, klikšķiniet **Jauns pakalpojumu pieprasījums** pa tālruni/e-pastu zemāk un atstājiet vaicājumu kā SharePoint migrācijas vairāk nekā 100 TB tekstlodziņā.

Pēc tam, kad iesniegta biļeti, lūdzu, atjauniniet to ar šādu informāciju: 

- Jūsu uzņēmuma nosaukums un īrnieks URL
- Aptuveno lielumu jūsu migrācijas
- Kad jūs gribētu sākt un pabeigt savu migrācijas novērtējums
- Aprakstīt, kur jums ir jūsu satura migrēšana no, piemēram, SharePoint serveris, rūtiņa, GDrive, failu koplietošanām, utt.
