# <a name="desktop-app-for-sharepoint"></a>Datora programma darbam ar SharePoint

Jūsu organizācija ir iekļauta programmas SharePoint datora lietojumprogrammas privātajā priekšskatījuma programmā.

Microsoft pašlaik neatbalsta datora lietojumprogrammu darbam ar SharePoint (privāts priekšskatījums). Lūdzu, izmantojiet savus Microsoft Private Preview partnerus, lai saņemtu palīdzību saistībā ar datora lietojumprogrammu darbam ar SharePoint.
