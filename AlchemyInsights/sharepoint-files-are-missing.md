# <a name="sharepoint-files-are-missing"></a>Trūkst SharePoint failu

Ja daudzi jūsu OneDrive vai SharePoint faili tiek izdzēsti, pārrakstīti, bojāti vai inficēti ļaunprogrammatūras, mēs lietotājiem sniedzam dažādas opcijas, lai palīdzētu atkopšanas procesā.

Papildinformāciju skatiet rakstā [trūkstošo, izdzēsto vai bojāto vienumu atkopšana pakalpojumā SharePoint Online un OneDrive darbam](https://go.microsoft.com/fwlink/?linkid=2110774).
