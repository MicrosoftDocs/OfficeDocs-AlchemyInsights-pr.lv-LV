# <a name="change-display-language-settings-in-sharepoint-or-onedrive"></a>Mainīt displeja valodu iestatījumus programmā SharePoint vai OneDrive 

Jūs nevarat mainīt noklusētos valodas iestatījumus SharePoint vai OneDrive, jo šie iestatījumi tika noteikta, veidojot abonementa Office 365. Tomēr var mainīt displeja valodu iestatījumus.

Lai izmainītu savus personīgos iestatījumus, skatiet [personas valodas un reģiona iestatījumu mainīšana](https://support.office.com/article/Change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7).

Lai mainītu portāla lietotāja interfeisu, sk. [Izvēlieties valodas, kuras vēlaties padarīt pieejamas vietnes lietotāja interfeisu](https://support.office.com/article/choose-the-languages-you-want-to-make-available-for-a-site-s-user-interface-16d3a83c-05ab-4b50-8fbb-ff576a3351e8).

Papildinformāciju par valodu iestatījumu Traucējummeklēšana, skatiet sadaļā [nevar mainīt noklusējuma valodu SharePoint Online vietņu kolekcijai](https://support.office.com/article/you-can-t-change-the-default-language-for-a-sharepoint-online-site-collection-40dda07e-6b41-49e9-9828-8805dcb92964).
