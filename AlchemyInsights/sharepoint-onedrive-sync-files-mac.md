# <a name="install-setup-and-sync-onedrive-or-sharepoint-files-on-mac"></a>OneDrive vai SharePoint failu instalēšana, iestatīšana un sinhronizēšana operētājsistēmā Mac 

Instalējiet, iestatiet un sinhronizējiet OneDrive vai SharePoint failus Mac datorā, izmantojot instrukcijas no [failu sinhronizēšanas ar onedrive sinhronizācijas klientu operētājsistēmā Mac OS X](https://support.office.com/article/sync-files-with-the-onedrive-sync-client-on-mac-os-x-d11b9f29-00bb-4172-be39-997da46f913f).

Ja ir instalēta programma OneDrive un iestatīšanas programma Mac datorā, bet lietotājiem rodas sinhronizācijas problēmas, skatiet sadaļu [sinhronizācijas problēmu novēršana pakalpojumā onedrive (Mac)](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756).

> [!NOTE]
> Zināmo mapju pārvietošanas līdzekli OneDrive sinhronizācijas sistēmai Windows nav pieejams Mac.
