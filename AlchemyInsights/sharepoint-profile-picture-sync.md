# <a name="when-do-my-profile-changes-sync-to-the-sharepoint-user-profile-application"></a>Kad mans profils maina sinhronizāciju ar SharePoint lietotāja profila lietojumprogrammu?

Profila attēlu sinhronizācijai pakalpojumā SharePoint Online ir nepieciešama lietotāja darbība, lai sāktu sava fotoattēla sinhronizēšanu. Nav automātiskas fotoattēlu sinhronizēšanas pakalpojumā SharePoint Online. SharePoint Online sinhronizē tikai fotoattēlus no Exchange Online, nevis Azure Active Directory.

Ja lietotājs nekad nav veicis darbību, kas uzaicinās veikt sinhronizāciju vai ir atjauninājis savu profila attēlu, un jaunais nav redzams pakalpojumā SharePoint Online, lietotājam, iespējams, būs jālūdz to sinhronizēt. Lūk, kā to var izdarīt:

1. Atveriet tīmekļa pārlūkprogrammu privātajā vai inkognito sesijā un [dodieties uz Office.com](https://www.office.com/).

2. Pierakstieties ar savu darba vai mācību iestādes kontu un noklikšķiniet uz elementa OneDrive vai SharePoint.

3. [Pievienojiet savu profila fotoattēlu pakalpojumā Microsoft 365](https://support.office.com/article/Add-your-profile-photo-to-Office-365-2eaf93fd-b3f1-43b9-9cdc-bdcd548435b7).

[Informācija par profila attēlu sinhronizāciju programmā Microsoft 365](https://support.office.com/article/Information-about-user-profile-synchronization-in-SharePoint-Online-177eb196-5887-43c9-84c3-b98a43d35129)
