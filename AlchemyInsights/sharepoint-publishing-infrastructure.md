# <a name="sharepoint-publishing-infrastructure"></a>SharePoint publicēšanas infrastruktūra


SharePoint ietver klasisku līdzekļu kopu, kas ļauj veidot un publicēt jūsu intranetā bagātīgas tīmekļa lapas. Aktivizējiet SharePoint publicēšanas infrastruktūru vietņu kolekcijas līmenī, lai tās ieslēgtu.

Lai iegūtu vairāk informācijas, lūdzu, apmeklējiet saites zemāk.

[Publicēšanas līdzekļu iespējošana](https://support.office.com/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518)

[Publicēšanas līdzekļi](https://support.office.com/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553)

Piezīmes:

- Klasiskās publicēšanas vietnes veidnes ietver publicēšanas līdzekļus pēc noklusējuma.

- Klasiskās publicēšanas funkcijas nav pieejamas modernās vietnēs. Mūsdienu vietnes pēc noklusējuma nodrošina jaunāku alternatīvu.
