# <a name="sharepoint-publishing-infrastructure"></a>SharePoint publicēšanas infrastruktūra


<p>SharePoint ietver līdzekļus, kas ļauj jums veidot un publicēt bagāts interneta lapas jūsu iekštīklam klasiskās kopu. SharePoint publicēšanas infrastruktūras vietņu kolekcijas līmenī, lai tos ieslēgtu aktivizēt.</p>  <p>Lai iegūtu vairāk informācijas, lūdzu, apmeklējiet zemāk esošajām saitēm.</p>  <ul>  <li><a href="https://support.office.com/en-us/article/Enable-publishing-features-479677A6-8B33-4AC7-907D-071C1C7E4518">Publicēšanas līdzekļu iespējošana</a></li>  <li><a href="https://support.office.com/en-us/article/Features-enabled-in-a-SharePoint-Online-publishing-site-3AB3810C-3C2C-4361-9D0E-0CBE666EA0B0?wt.mc_id=O365_Portal_MMaven#__toc336865553">Publicēšanas līdzekļi</a></li>  </ul>  <p>Piezīmes:</p>  <ul>  <li>Klasisks publicēšanas vietņu veidnēs ir publicēšanas līdzekļi pēc noklusējuma.</li>  <li>Klasisks publicēšanas līdzekļi nav pieejami mūsdienu vietnēs. Mūsdienu vietnes piedāvā jaunākās alternatīvu pēc noklusējuma.</li>  </ul>
