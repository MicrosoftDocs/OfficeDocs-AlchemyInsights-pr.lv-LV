# <a name="sharepoint-online-throttling"></a>SharePoint tiešsaistē droselēšanas

SharePoint Online lieto droselēšanas uzturēt optimālu veiktspēju un uzticamību SharePoint Online pakalpojumiem. Droselēšanas ierobežojumus lietotāja darbībām vai vienlaicīgā pieslēguma numurs aicina (ar skriptu vai kods), lai izvairītos no resursu pārmērīgas. Lai iegūtu vairāk informācijas, lūdzu apmeklējiet šīs saites.

- [Izvairītos no mitināts vai bloķēts SharePoint Online](https://docs.microsoft.com/en-us/sharepoint/dev/general-development/how-to-avoid-getting-throttled-or-blocked-in-sharepoint-online)

- [Datu migrēšanas un SPO droselēšanas](https://blogs.technet.microsoft.com/sposupport/2017/08/12/data-migration-and-spo-service-throttling/)

- [SharePoint tiešsaistē un OneDrive migrācijas ātrums](https://docs.microsoft.com/en-us/sharepointmigration/sharepoint-online-and-onedrive-migration-speed)

 - [Apstrādāt, izmantojot eksponenciālās atlēca droselēšanas SharePoint Online](https://docs.microsoft.com/en-us/sharepoint/dev/solution-guidance/handle-sharepoint-online-throttling-by-using-exponential-back-off)

- [Ražīguma plānošana un slodzes testēšana SharePoint Online](https://support.office.com/en-us/article/Capacity-planning-and-load-testing-SharePoint-Online-c932bd9b-fb9a-47ab-a330-6979d03688c0)
