# <a name="sharing-options-appear-dim-or-are-not-available"></a>Kopīgošanas opcijas izskatās blāvas vai nav pieejamas

Ja dažas koplietošanas opcijas vai nav pieejamas, mēģinot koplietot no SharePoint Online vai OneDrive, skatiet [koplietošanas opcijas ir pelēkota, koplietojot SharePoint Online vai onedrive](https://docs.microsoft.com/sharepoint/support/administration/sharing-options-grayed-out-when-sharing-from-sharepoint-online-or-onedrive).
