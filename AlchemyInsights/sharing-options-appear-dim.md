# <a name="sharing-options-appear-dim-or-are-not-available"></a>Koplietošanas opcijas izskatās blāvas vai nav pieejamas

Ja dažas kopīgošanas opcijas vai nav pieejamas, mēģinot koplietot no SharePoint Online vai OneDrive, skatiet rakstu [koplietošanas opcijas ir pelēkotas, koplietojot no SharePoint Online vai OneDrive](https://docs.microsoft.com/sharepoint/support/administration/sharing-options-grayed-out-when-sharing-from-sharepoint-online-or-onedrive).
