# <a name="site-is-not-available"></a>Vietne nav pieejama

Ja jūsu SharePoint vai OneDrive vietā nav pieejams vairākiem lietotājiem, var būt pagaidu pakalpojumu jautājums. Pārbaudiet[Pakalpojumu veselības Dashboard](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) redzēt, ja jūsu īrnieks ir ietekmētas.
