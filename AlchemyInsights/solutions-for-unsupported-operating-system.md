# <a name="solutions-for-unsupported-operating-system"></a>Risinājumi "neatbalstīta operētājsistēma"

Mēģinot instalēt Microsoft 365, var tikt parādīts **neatbalstīts operētājsistēmas** kļūdas ziņojums, ja jūsu ierīce neatbilst [sistēmas prasībām](https://products.office.com/office-system-requirements).
  
Microsoft 365 ir nepieciešams:
  
- PC datorā: Windows 10, Windows 8,1, Windows 8, Windows 7 1. servisa pakotne, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 vai Windows Server 2008 R2

    VAI

- Mac OS X 10,10

Ja izveidojāt šo ziņojumu, mēģinot instalēt Office mobilajā ierīcē vai Chromebook datorā, skatiet rakstu [kā lokāli instalēt un palaist Microsoft Office](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
