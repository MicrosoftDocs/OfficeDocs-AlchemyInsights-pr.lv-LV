# <a name="solutions-for-unsupported-operating-system"></a>Risinājumus "Neatbalstītai operētājsistēmai"

Mēģinot instalēt Microsoft 365, iespējams, redzēsit **neatbalstītu operētājsistēmas** kļūdas ziņojumu, ja jūsu ierīce neatbilst [sistēmas prasībām](https://products.office.com/office-system-requirements).
  
Microsoft 365 nepieciešams:
  
- PC: Windows 10, Windows 8,1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 vai Windows Server 2008 R2

    Vai

- Mac OS X 10,10

Ja šis ziņojums tika parādīts, mēģinot instalēt Office mobilajā ierīcē vai Chromebook datorā, skatiet sadaļu [kā instalēt un palaist Microsoft Office chromebook](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA)datorā.
