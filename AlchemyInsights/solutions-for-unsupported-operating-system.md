# <a name="solutions-for-unsupported-operating-system"></a>"Neatbalstīta operētājsistēma" risinājumi



Mēģinot instalēt Office 365, **neatbalstīta operētājsistēma** kļūdas ziņojumu var redzēt, ja jūsu ierīce neatbilst [sistēmas prasībām](https://products.office.com/office-system-requirements).
  
Biroja 365 prasa:
  
- PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, 2016. Windows Server, Windows Server 2012 R2, Windows Server 2012 vai Windows Server 2008 R2
    
    OR
    
- Mac OS X 10.10
    
Ja saņemts šāds ziņojums, mēģinot instalēt Office uz mobilo ierīci vai Chromebook, uzzinātu, [kā instalēt un palaist Microsoft Office Chromebook](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
