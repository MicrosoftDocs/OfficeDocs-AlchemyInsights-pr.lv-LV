# <a name="supported-version-of-the-sharepoint-migration-tool"></a>SharePoint migrācijas rīka atbalstītā versija



Pašlaik SharePoint migrēšanas rīks atbalsta tikai migrēšanu no SharePoint 2013. Pašlaik netiek atbalstītas versijas 2010 un 2016.
  
Papildinformāciju skatiet rakstā [SharePoint migrācijas rīka ieviešana](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
