# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Atbalstīta SharePoint migrācijas rīka versiju



Pašlaik SharePoint migrēšanas rīks atbalsta tikai migrāciju no SharePoint 2013. Versijas 2010 un 2016 netiek atbalstīti šajā laikā.
  
Lai iegūtu papildu informāciju, skatiet [SharePoint migrēšanas rīku, ar ko ievieš](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
