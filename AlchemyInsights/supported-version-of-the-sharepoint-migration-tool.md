# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Atbalstītā SharePoint migrācijas rīka versija



Pašlaik SharePoint migrācijas rīks atbalsta tikai migrēšanu no SharePoint 2013. Versijas 2010 un 2016 pašlaik netiek atbalstītas.
  
Lai iegūtu papildinformāciju, skatiet [iepazīšanās ar SharePoint migrācijas rīku](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
