# <a name="what-you-can-accomplish-with-microsoft-teams-powershell-module"></a>Ko varat paveikt, izmantojot Microsoft Teams PowerShell moduli

Ja vēlaties uzzināt, kā instalēt Teams PowerShell moduli, varat to apskatīt [šajā emuārā](https://blogs.technet.microsoft.com/skypehybridguy/2017/11/07/microsoft-teams-powershell-support/).

Papildinformācija:

- Pārskatiet [Teams PowerShell pārskatu](https://docs.microsoft.com/MicrosoftTeams/teams-powershell-overview) , lai izprastu [administratora lomu iespējas](https://docs.microsoft.com/MicrosoftTeams/using-admin-roles), [veidotu un pārvaldītu grupas, izmantojot PowerShell](https://docs.microsoft.com/MicrosoftTeams/teams-powershell-overview#creating-and-managing-teams-via-powershell), un [pārvaldītu politikas](https://docs.microsoft.com/MicrosoftTeams/teams-powershell-overview#managing-policies-via-powershell) vai [konfigurācijas](https://docs.microsoft.com/MicrosoftTeams/teams-powershell-overview#managing-configurations-via-powershell). 
- Pārskatiet [Microsoft Teams PowerShell komandu atsauci](https://docs.microsoft.com/powershell/module/teams/?view=teams-ps) uz komandu sarakstu, kas pieejama, izmantojot PowerShell.
