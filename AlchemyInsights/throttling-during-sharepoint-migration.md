# <a name="sharepoint-migration-throttling"></a>SharePoint migrācijas ierobežošana

SharePoint Online izmanto ierobežošanu, lai uzturētu optimālu SharePoint Online pakalpojuma veiktspēju un uzticamību. Ierobežošana ierobežo lietotāju darbību vai vienlaicīgu zvanu skaitu (ar skriptu vai kodu), lai nepieļautu pārmērīgu resursu pielietojumu. Lai iegūtu vairāk informācijas, lūdzu, skatiet:

- [Migrēšanas laikā radās vāja veiktspēja vai ierobežošana](https://docs.microsoft.com/sharepointmigration/sharepoint-online-and-onedrive-migration-speed#faq-and-troubleshooting)
