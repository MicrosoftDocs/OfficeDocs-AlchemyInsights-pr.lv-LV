# <a name="gatewaynotreachable-when-setting-credentials"></a>GatewayNotReachable, iestatot akreditācijas datus

Jūs varat saskarties ar GatewayNotReachable, mēģinot iestatīt akreditācijas datus datu avotam pakalpojumā Power BI. Tas varētu būt novecojis vārtejas rezultāts. Instalējiet jaunāko vārteju un mēģiniet vēlreiz.

Lai iegūtu vairāk informācijas:[https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
