# <a name="gatewaynotreachable-when-setting-credentials"></a>GatewayNotReachable, nosakot akreditācijas datus

GatewayNotReachable var rasties, mēģinot iestatīt akreditācijas datus datu avotā. Tas varētu būt novecojis vārtejas rezultāts. Instalējiet jaunāko vārteju un mēģiniet vēlreiz.
