# <a name="troubleshooting-outlook-mobile-setup-for-a-shared-mailbox"></a>Outlook Mobile iestatīšanas problēmu novēršana koplietotai pastkastei

Koplietojamai pastkastei nevar piekļūt no programmas Outlook, kas ir instalēta mobilajā ierīcē. Vai vēlaties šo līdzekli? Ļaujiet programmai Outlook inženieru komandai. [Balsojiet šeit](https://go.microsoft.com/fwlink/?linked=862116)!
  
Jūs varat izmantot koplietojamu pastkasti mobilajā ierīcē, atverot pārlūkprogrammu, pierakstoties un pēc tam dodoties uz Outlook tīmeklī. Norādījumus skatiet rakstā [koplietojamas pastkastes pievienošana programmā Outlook tīmeklī](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207). No Outlook tīmeklī jūs varēsit piekļūt koplietojamai pastkastei.
  
Papildinformācija: [Saņemiet palīdzību par programmu Outlook darbam ar iOS un Android](https://support.office.com/article/Get-in-app-help-for-Outlook-for-iOS-and-Android-218a22d1-9fa5-4889-b689-de1c63493243)
