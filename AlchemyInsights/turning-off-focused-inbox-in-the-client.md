# <a name="turn-off-focused-inbox-in-email-clients"></a>Fokusēta iesūtnes izslēgšana e-pasta klientos

Lai izslēgtu fokusēta Iesūtne programmā Outlook 2016, Outlook Web, Outlook.com/Hotmail.com, Outlook 2016 For Mac vai Mail for Windows 10, skatiet sadaļu [fokusēta iesūtnes](https://support.office.com/article/f714d94d-9e63-4217-9ccb-6cb2986aa1b2.aspx)izslēgšana.

Lai izslēgtu fokusēta Iesūtne ikvienam organizācijas lietotājam, skatiet sadaļu [fokusēta iesūtnes konfigurēšana visiem jūsu uzņēmuma](https://docs.microsoft.com/office365/admin/setup/configure-focused-inbox)darbiniekiem.
