# <a name="unable-to-add-an-attachment-to-an-externally-facing-form"></a>Nevar pievienot pielikumu ārēji novietotai formai

Ja mēģināt pievienot pielikumu ārējai Microsoft veidlapu aptaujai, šis līdzeklis vēl nav pieejams. Lai izsekotu šī līdzekļu pieprasījuma statusu, dodieties uz [šo UserVoice ierakstu](https://go.microsoft.com/fwlink/?linkid=2133069)un pievienojiet savu balsojumu.
