# <a name="update-taxvat-id"></a>PVN/PVN ID atjaunināšana

Ja jums ir problēmas ar nodokļu/PVN ID, sazinieties ar Microsoft licencēšanas partneri (partnera vai programmatūras padomnieks). Tās sazināsies ar reģionālo operāciju centru (ROC), lai saņemtu jautājumus par jūsu nodokļu/PVN ID.
