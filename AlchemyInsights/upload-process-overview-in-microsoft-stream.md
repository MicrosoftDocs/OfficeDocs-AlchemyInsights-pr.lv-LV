# <a name="upload-process-overview-in-microsoft-stream"></a>Augšupielādes procesa apskats programmā Microsoft Stream

Ikreiz, kad augšupielādējat videoklipu, Microsoft Stream automātiski kodē to vairākos bitu pārraides ātrumus, lai video varētu straumēt, izmantojot adaptīvo bitrate Streaming. Microsoft Stream optimizē atskaņošanas video bitu pārraides ātrumu un kvalitāti, pamatojoties uz skatītāja pieejamo joslas platumu.

- [Augšupielādes procesa apskats programmā Microsoft Stream](https://docs.microsoft.com/stream/upload-process-overview)
