# <a name="your-users-mailbox-is-being-created"></a>Jūsu lietotāja pastkasti tiek veidots

Tas var aizņemt kādu stundu pirms jauna pastkaste ir gatava lietošanai.
  
Ja lietotāja pastkastē ir bijis "gatavo" ilgāk par stundu, jūs varat mēģināt dzēšana un jāveido jauna lietotāju vai izmantojiet opciju "Ļaujiet mums piezvanīt jums" runāt ar atbalsta inženieri.
