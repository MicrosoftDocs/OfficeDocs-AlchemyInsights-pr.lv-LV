# <a name="using-a-shared-mailbox-on-a-mobile-device"></a>Izmantojot koplietojamās pastkastes mobilajā ierīcē

Nevar piekļūt koplietojuma pastkastes no Outlook app, kas ir instalēta mobilajā ierīcē. Vai vēlaties šo līdzekli? Ļaujiet Outlook inženieru komanda, kas zina. [Balsojums šeit](https://go.microsoft.com/fwlink/?linked=862116)!
  
Atverot pārlūku, pierakstīšanās pakalpojumā Office 365 un tad dodas uz Outlook Web, var izmantot tikai koplietojamās pastkastes mobilajā ierīcē. Instrukcijas skatiet [Add koplietojamo pastkasti programmā Outlook izveidojāt Web](https://support.office.com/article/add-a-shared-mailbox-to-outlook-on-the-web-98b5a90d-4e38-415d-a030-f09a4cd28207).
  
Ja jāpiekļūst koplietojamo pastkastēm, izmantojot programmu Outlook app mobilajā ierīcē, apsveriet iespēju izveidot Office 365 grupa, tā vietā, lai lietotu koplietojamu pastkasti. Plašāku informāciju skatiet [salīdzināt personu grupas](https://support.office.com/article/758759ad-63ee-4ea9-90a3-39f941897b7d.aspx).
