# <a name="why-cant-i-edit-this-file"></a>Kāpēc nevar rediģēt šo failu?

Ja dokuments, darblapa vai prezentācija ir bloķēta rediģēšanai, un to nevar rediģēt Windows datora lietojumprogrammās, iespējams, ka failu rediģē cits lietotājs, vai arī fails ir paņemts.

Ātri aplūkojiet dažus iemeslus, kādēļ, [iespējams, nevarēsit rediģēt failu](https://support.office.com/article/why-can-t-i-edit-this-file-97315f48-aa5e-49d3-a4ae-a14b73daf87b).
