# <a name="windows-fresh-start"></a>Windows Fresh Start

Windows Fresh Start ierīces darbība piedāvā alternatīvu ierīces tīrīšanas un ierīces darbības pārtraukšanas darbībām, kā arī opciju saglabāt ierīces reģistrācijas stāvokli kopā ar lietotāja mājas mapes programmām un iestatījumiem.

Lai iegūtu papildinformāciju, skatiet rakstu [jaunas sākuma lietošana, lai atiestatītu Windows 10 ierīces ar Intune](https://docs.microsoft.com/intune/device-fresh-start).
