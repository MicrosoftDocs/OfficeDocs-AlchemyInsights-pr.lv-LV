# <a name="yammer---possible-service-issue"></a>Yammer — iespējamās pakalpojumu problēmas

Lai piekļūtu pakalpojumam Yammer, var rasties īslaicīga problēma. Pārbaudiet [Pakalpojuma darbspējas informācijas paneli](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) , lai uzzinātu, vai tas ietekmē jūsu tīklu.
